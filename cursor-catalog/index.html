Windows XP 桌面引擎 — 真实使用的鼠标样式汇总

本页汇总了当前项目（src/）中真实使用到的全部鼠标指针样式。 每个样式均已转换为 PNG 并分配唯一编号，同时标注对应的 CSS cursor 角色与实际使用场景。

数据来源： src/assets/cursors/xp/ 目录下的 14 个 .cur 文件， 这些资源在 src/scoped.css 中通过 cursor: url(...) 映射到具体 CSS 角色。 此外，cursor: none 用于需要隐藏鼠标的特殊场景，也作为独立条目列出。

总数：14 个真实使用的鼠标样式（13 个图片 + 1 个透明/隐藏样式）。
XP-CUR-001 default
arrow_r.cur → arrow_r.png
热点：(0, 0)
使用场景： 整个 .windows-xp-root 范围的默认鼠标指针；未覆盖其他光标样式时的缺省指针。
XP-CUR-002 pointer
harrow.cur → harrow.png
热点：(12, 1)
使用场景： 高对比度 / 辅助功能默认手型链接指针（来源：windows-ui-assets/Cursors/Windows XP/harrow.cur）。 用于链接（<a>）、按钮（<button>）、[role="button"]、 .xp-clickable 等全部可点击元素；项目中最常用的交互指针。
XP-CUR-003 text

热点：(15, 16)
使用场景： 输入框（<input>）、文本域（<textarea>）、 [contenteditable] 以及命令提示符等需要文本输入的光标区域。
