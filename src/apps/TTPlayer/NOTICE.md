# 千千静听 TTPlayer 致谢与引用 / Attribution & references

本应用（`src/apps/TTPlayer/`）是向 2002–2007 年中文互联网经典音乐播放器
**千千静听（TTPlayer）** 的自绘致敬。布局与交互参考了开源网页复刻项目
**[haixiangyan/ttplayer](https://github.com/haixiangyan/ttplayer)**
（用 React + Audio API 实现的千千静听）。

This app (`src/apps/TTPlayer/`) is a self-drawn homage to **TTPlayer
(千千静听)**, the classic 2002–2007 Chinese music player. Its layout and
interactions reference the open-source web remake
**[haixiangyan/ttplayer](https://github.com/haixiangyan/ttplayer)** (a TTPlayer
built with React + the Audio API).
