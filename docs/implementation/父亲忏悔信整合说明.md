### 方案B：支持外部文件加载（推荐用于生产环境）

修改代码支持从外部文件动态加载内容。

**需要修改的文件：**

1. **FileSystemContext.jsx** - 添加对 `contentPath` 的支持
2. **Notepad.jsx** - 支持异步加载内容

**实现示例：**

```javascript
// 在 FileSystemContext.jsx 中
const loadFileContent = async (contentPath) => {
  try {
    const response = await fetch(contentPath);
    const text = await response.text();
    return text;
  } catch (error) {
    console.error('Failed to load file content:', error);
    return '文件加载失败';
  }
};

// 在打开文件时
if (node.contentPath) {
  const content = await loadFileContent(node.contentPath);
  // 使用 content
} else {
  // 使用 node.content
}
```

### 方案C：使用 import.meta.glob（最优雅）

利用 Vite 的 `import.meta.glob` 功能动态导入文本文件。

**实现示例：**

```javascript
// 在 FileSystemContext.jsx 中
const documents = import.meta.glob('../data/documents/*.txt', {
  as: 'raw',
  eager: false
});

// 使用时
const loadDocument = async (filename) => {
  const path = `../data/documents/${filename}`;
  if (documents[path]) {
    return await documents[path]();
  }
  return null;
};
```
