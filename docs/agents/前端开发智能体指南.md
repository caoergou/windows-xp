#### 任务 FE-02：增强加密文件夹功能
**预计时间**：0.5天
**优先级**：⭐⭐⭐
**依赖**：FE-01

**需求描述**：
在文件管理器中，点击加密文件夹时弹出密码输入框，验证通过后才能打开。

**修改文件**：`src/apps/Explorer.jsx`

**实现步骤**：
1. 检测文件夹是否有 `locked: true` 属性
2. 如果加密，显示锁图标
3. 点击时调用 PasswordDialog
4. 验证密码（使用 FileSystemContext 的 checkAccess）
5. 验证通过后正常打开文件夹

**代码示例**：
```javascript
const handleFolderClick = (folder) => {
  if (folder.locked) {
    showPasswordDialog({
      correctPassword: folder.password,
      onSuccess: () => {
        // 打开文件夹
        navigateToFolder(folder);
      }
    });
  } else {
    navigateToFolder(folder);
  }
};
```

**验收标准**：
- [ ] 加密文件夹显示锁图标
- [ ] 点击加密文件夹弹出密码框
- [ ] 输入正确密码可以打开
- [ ] 输入错误密码无法打开
- [ ] 非加密文件夹正常打开
