### 任务 DA-03：设计EXIF元数据格式
**预计时间**：0.5天
**优先级**：⭐⭐⭐

**需求描述**：
设计照片EXIF元数据的JSON格式。

**输出文件**：`docs/data-specs/exif-format.md`

**数据结构设计**：

```json
// src/data/photos/evidence_001_exif.json
{
  "fileName": "evidence_001.jpg",
  "fileSize": "2.3 MB",
  "dimensions": "4272 x 2848",
  "camera": {
    "make": "Canon",
    "model": "EOS 600D",
    "lens": "EF-S 18-55mm f/3.5-5.6 IS II"
  },
  "settings": {
    "aperture": "f/2.8",
    "shutterSpeed": "1/250",
    "iso": "400",
    "focalLength": "35mm",
    "exposureMode": "Manual",
    "whiteBalance": "Auto"
  },
  "dateTime": {
    "original": "2015:11:23 14:30:15",
    "digitized": "2015:11:23 14:30:15",
    "modified": "2015:11:23 14:30:15"
  },
  "location": {
    "description": "云山县一中",
    "gps": null
  },
  "notes": "这张照片的曝光三角参数是：2.8-1/250-400"
}
```

**关键字段说明**：
- `aperture`: 光圈值（用于密码）
- `shutterSpeed`: 快门速度（用于密码）
- `iso`: ISO感光度（用于密码）
- `dateTime.original`: 拍摄时间（用于时间线验证）

**验收标准**：
- [ ] 包含所有必要的EXIF字段
- [ ] 支持密码线索
- [ ] 时间格式正确
- [ ] 包含注释说明
