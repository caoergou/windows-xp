# 重写 puzzle_hints.json

**类别**：A类
**优先级**：高
**依赖**：无
**修改文件**：`src/data/puzzle_hints.json`

## 背景

puzzle_hints.json 是一个早期版本残留，其中所有谜题的密码和线索来源与当前设计文档（docs/解密.md）及实际游戏数据（filesystem.json、albums.json、encrypted_diary.json）完全矛盾。如果玩家触发提示系统，会得到错误的密码，导致游戏流程卡死。

## 具体修改内容

将 puzzle_hints.json 中全部5个谜题的密码、线索来源、渐进提示文本，按以下对照表重写：

| 谜题 | 正确密码 | 正确线索来源 |
|------|---------|------------|
| 相册密码 | `28125400` | 群聊2015.11.20"光之公式"对话（f/2.8, 1/125, ISO400） |
| 父亲第一层 | `19990315` | README.txt 提示"密码是你的生日"，夏灯生日 |
| 加密日志 | `camera3rdeye` | QQ空间加密日志预览区白色隐写文字"Camera is the 3rd eye" |
| 现代诗藏头 | 无密码（内容谜题） | 群聊2015.12.28 林晓宇提到"每行第三个字连起来读" |
| 父亲第二层 | `20151123` | 工作日志2015.11.23条目"从今天起我的灯熄灭了" |

每个谜题的渐进提示按以下阈值设置：
- 核心谜题（相册、加密日志、父亲第二层）：5/8/12/15/20次尝试后逐级解锁
- 辅助谜题（父亲第一层）：3/5/10次尝试后解锁

提示内容不得在阶段二的提示中泄露阶段三的信息（如陈默动机、完整短信内容）。

## 验收标准

- puzzle_hints.json 中所有密码与 filesystem.json、albums.json、encrypted_diary.json 中的实际密码完全一致
- 每个谜题的线索来源指向游戏中真实存在的素材
- 核心谜题与辅助谜题的提示阈值已区分
- 提示文本不包含跨阶段信息泄露
