# 修正 photo_001_exif.json 相机参数

**类别**：A类
**优先级**：高
**依赖**：无
**修改文件**：`src/data/photos/evidence/photo_001_exif.json`

## 背景

photo_001_exif.json 当前使用了错误的相机型号（Sony DSC-W830）和错误的拍摄参数（f/4.5, 1/250），导致玩家即使按设计意图查看 EXIF 数据，也无法推导出相册密码 28125400。这是谜题1线索链的阻断性错误。

## 具体修改内容

将 photo_001_exif.json 中以下字段修改为：

| 字段 | 当前值（错误） | 修改为 |
|------|-------------|--------|
| 相机型号 | Sony DSC-W830 | Canon EOS 550D |
| 光圈值 | f/4.5 | f/2.8 |
| 快门速度 | 1/250 | 1/125 |
| ISO感光度 | 400 | 400（不变） |
| 用户注释 | （如有）任意值 | Camera is the 3rd eye |

用户注释字段"Camera is the 3rd eye"是林晓宇手动写入的，与群聊中父亲背带刻字、QQ空间个性签名形成三重呼应，是密码发现路径的关键节点。

## 验收标准

- photo_001_exif.json 中相机型号为 Canon EOS 550D
- 光圈值为 f/2.8，快门速度为 1/125，ISO 为 400
- 用户注释字段包含"Camera is the 3rd eye"
- 玩家通过查看该 EXIF 文件，可以推导出密码 28125400（f/2.8 → 28，1/125 → 125，ISO400 → 400，拼接为 28125400）
