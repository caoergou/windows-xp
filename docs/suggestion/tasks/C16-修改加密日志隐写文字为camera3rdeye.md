## 解锁内容

### 现代诗《第三只眼》

（完整文本见 task-09，此处为引用）

```
王朝的月光照不进这条街的尽头
虎皮的鼓声在远处隆隆作响
替我说话的风也沉默了
考场外的梧桐叶落了又落
我站在楼顶看见整座城市的灯
被夜色一盏一盏地熄灭
杀死沉默的，只有更深的沉默
```

首字连读：王虎替考我被杀（谜题4的内容）

### 日志正文第一部分

```
（如果你能看到这里，说明你找到了相机。试着用相机的眼光读这首诗——第一眼看到的，往往是最重要的。）

陈默告诉我他为什么要做这件事。
他奶奶需要手术，三万块。
他家里没有。
王虎找到他，说只要帮忙改几条记录，钱就有了。

我理解他。
我真的理解他。
但我不能假装没看见。

我把那天晚上的系统日志截图保存了一份，放在夏叔叔的电脑上（他不知道）。
那是我唯一能用的电脑——我自己的电脑被陈默警告过，我不敢存在上面。
如果有一天我出事了，去回收站找找。
被删除的东西，不一定真的消失了。

陈默说他想陪我去。
他说他想保护我。
我不知道他说的是真的还是假的。
我不知道他是想保护我，还是想阻止我。
```
