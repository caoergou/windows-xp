# 新增阶段四山顶照片情感落点

**类别**：C类
**优先级**：中
**依赖**：依赖 C04（《给夏灯.txt》修改完成后，照片作为补充情感落点）
**修改文件**：
- `src/data/filesystem.json`（在父亲文件夹第二层添加照片文件节点）
- `src/data/photos/`（新建照片描述文件）

## 背景

阶段四需要一个让玩家久久无法平静的情感落点。在墓前重聚之前，夏灯在整理父亲遗物时发现一张三人山顶合照，背面有林晓宇的字迹。这张照片没有叙事功能，它只是让玩家记住：在所有这些悲剧发生之前，他们曾经是朋友。

## 具体修改内容

**数据层**：

在 filesystem.json 中，父亲文件夹第二层（爸爸的信文件夹）内添加一个照片文件节点：

```json
{
  "type": "file",
  "name": "山顶合照.jpg",
  "icon": "photo",
  "description": "2015年9月，山顶，三个少年。林晓宇在拍照，陈默在看镜头，夏灯在笑。照片背面有林晓宇的字迹：'山顶事务所，成立纪念。记录、守护、真相。——2015.09.15'"
}
```

**照片描述文件**：

新建 `src/data/photos/mountain_top_photo.json`：

```json
{
  "filename": "山顶合照.jpg",
  "taken_by": "林晓宇（自拍）",
  "taken_at": "2015-09-15",
  "description": "2015年9月，山顶，三个少年。林晓宇在拍照，陈默在看镜头，夏灯在笑。",
  "back_note": "山顶事务所，成立纪念。记录、守护、真相。——2015.09.15",
  "found_by": "夏建国，从学校失物招领处取回，一直放在抽屉里，从未告诉任何人。"
}
```

照片的 found_by 字段说明：夏父在林晓宇死后，从学校的失物招领处拿走了这张照片，一直放在抽屉里，没有告诉任何人。这个细节说明夏父在沉默的九年里，一直记得林晓宇。他的沉默不是冷漠，而是无力。

## 验收标准

- filesystem.json 父亲文件夹第二层包含"山顶合照.jpg"文件节点
- 照片描述包含三人信息和林晓宇的字迹内容
- found_by 字段说明夏父保存照片的细节
- 照片在游戏中可通过图片查看器打开
- 照片时间戳为2015年9月15日（与时间线.md 山顶事务所成立日期一致）
