# 新增陈默邮件5（替代草稿箱方案）

**类别**：B类
**优先级**：高
**依赖**：依赖 B05（完整短信记录需先存在，邮件5才能引用）
**修改文件**：`src/data/email/inbox/email_chenmo_005.txt`（修改现有文件内容）

## 背景

阶段三需要陈默通过邮件坦白2月15日现场所见和九年等待的心理状态。原提议中的"草稿箱"方案存在逻辑漏洞（草稿保存在陈默本地设备，不可能出现在夏父电脑），创作优化总方案已明确以邮件5替代。

## 具体修改内容

将 email_chenmo_005.txt 的内容替换为以下文本：

```
夏灯，

我写了很多遍，删了很多遍，最后决定直接发给你。

2月15日那天，我提前到了烂尾医院。
我看到赵威他们在等着。
我知道他们要做什么。

我想拦住他们，但我没有。
我躲在角落里，看着这一切发生。
我听到了声音。然后就没有了。

我是懦夫。我一直是懦夫。

但我保存了所有证据。
156页的操作日志，47条短信，3张截图，还有我自己写的陈述。

九年了，每年2月15日，我都会打开那个文件夹，看一遍，然后关上。
我不知道我在等什么。
也许是等一个人来问我。

你来了。

陈默
```

邮件触发时机：玩家解锁加密日志《第三只眼》后触发，或玩家看完林晓宇日志正文后触发。

## 验收标准

- email_chenmo_005.txt 包含"我写了很多遍，删了很多遍"的开头
- 包含陈默在现场目睹事件的坦白
- 包含"九年了，每年2月15日"的等待描述
- 以"你来了"结尾，完成从"调查者"到"见证者"的身份转变
- 不包含任何草稿箱相关的内容
