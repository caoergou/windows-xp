# 新增学校官网历史快照系统腐败种子

**类别**：C类
**优先级**：低
**依赖**：无
**修改文件**：`src/data/`（学校官网相关数据文件，具体路径取决于 InternetExplorer 组件的数据组织方式）

## 背景

在模拟浏览器中，玩家可以访问县一中官网历史快照。在2015年9月的公告栏中，有一条不起眼的通知，其中"如有学籍信息需要更正，请联系教务处夏主任"这句话在阶段一读来只是普通行政通知，在阶段四读来有了完全不同的含义——这条通知发布的时间，正是陈默开始帮王虎篡改学籍数据的前夕。

## 具体修改内容

在学校官网历史快照数据中，在2015年9月的公告栏添加以下通知：

```
关于2015-2016学年新生学籍注册的通知

各位同学，请于9月20日前完成学籍注册。
如有学籍信息需要更正，请联系教务处夏主任。

县一中教务处
2015年9月10日
```

注意：
- 通知发布时间为2015年9月10日，陈默开始帮王虎篡改数据是2015年11月23日，时间差合理
- "联系教务处夏主任"中的"夏主任"即夏建国，玩家在阶段四才会意识到这个联系
- 通知内容普通，不会在阶段一引起玩家特别注意

## 验收标准

- 学校官网2015年9月快照中包含上述通知
- 通知在游戏中可通过模拟浏览器访问学校官网查看
- 通知内容不包含任何明显的"可疑"标记，保持普通行政通知的外观
