## 解密路径

1. 玩家在阶段一看到群聊"光之公式"对话，林晓宇明确说"纯数字部分拼在一起"
2. 玩家访问林晓宇QQ空间，发现加密相册，描述提示"三个数字，看看照片属性"
3. 玩家下载相册封面图片到本地
4. 右键图片 → 属性 → 摘要 → 高级，看到EXIF数据：
   - 光圈：f/2.8
   - 快门：1/125
   - ISO：400
5. 回忆群聊中"纯数字部分拼在一起"：28 + 125 + 400 = `28125400`
6. 在相册密码框输入 `28125400`，解锁成功
