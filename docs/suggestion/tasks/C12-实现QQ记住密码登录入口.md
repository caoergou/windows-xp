# task-01：QQ登录入口设计

谜题编号：0（前置，不作为谜题）
难度：无
所属阶段：阶段一前置
预计玩家用时：1分钟

---

## 设计说明

QQ登录不是谜题，是游戏的入口。夏灯以前用过这台电脑，QQ客户端勾选了"记住密码"，账号和密码已自动填入，玩家直接点击登录即可。

这个设计传递的信息：时间在这台电脑上停住了。QQ还记得夏灯，就像父亲还记得她一样。

---

## 技术实现

### 登录界面状态

- 账号栏：已填入（夏灯的QQ号）
- 密码栏：已填入（显示为 ●●●●●●●●，实际值 `19990315`）
- "记住密码"复选框：已勾选
- "自动登录"复选框：未勾选（需要玩家主动点击，感受到"我选择进入"）

### 玩家操作

点击"登录"按钮，无需输入任何内容。

## 情感设计

玩家登录后，看到林晓宇的头像旁显示"最后登录：2016年2月14日"——那是他坠楼前一天。

调查笔记自动弹出，夏灯写下：

```
晓宇，对不起，我来晚了。
```

这句话奠定了整个游戏的情感基调：不是侦探破案，而是一个人在追寻她本不该错过的真相。
