# 补充浏览器历史记录（夏父十年心路）

**类别**：B类
**优先级**：中
**依赖**：无
**修改文件**：`src/data/filesystem.json`（IE浏览器历史记录相关节点）

## 背景

IE浏览器历史记录可以成为展示夏建国十年心路历程的时间线工具。当前历史记录内容不详，需要补充从2016年到2025年的关键搜索记录，让玩家在阶段四看到夏父不是冷漠的旁观者，他一直在挣扎，只是每次都退缩了。

## 具体修改内容

在 filesystem.json 中，找到 IE 浏览器历史记录的数据节点，添加以下条目（按时间倒序排列）：

```
[2025-12-28] 百度搜索："肝癌晚期存活时间"
[2025-12-20] 百度搜索："如何向纪检委举报"
[2025-10-15] 新闻页面："王虎团伙被一锅端，涉案金额逾千万"
[2020-03-12] 新闻页面："王虎因金融诈骗被捕"
[2016-02-20] 百度搜索："意外坠楼死亡认定标准"
[2016-02-16] 百度搜索："举报人保护制度"
[2016-02-16] 百度搜索："匿名举报渠道"
```

这些搜索记录构成夏建国十年心路历程的缩影：
- 2016年2月16日（林晓宇死后第二天）：他在搜索举报渠道，但最终没有举报
- 2020年3月：王虎被抓，他关注了新闻，但没有行动
- 2025年10月：王虎团伙被端，他再次关注
- 2025年12月：确诊癌症，搜索存活时间，同时搜索举报渠道——这次他决定留下证据给女儿

## 验收标准

- filesystem.json 中 IE 浏览器历史记录包含上述7条条目
- 条目按时间倒序排列（最新的在最前面）
- 2016年2月16日的两条搜索记录时间与夏父草稿删除时间一致（B10任务）
- 历史记录在游戏中可通过 IE 浏览器的历史记录功能查看
