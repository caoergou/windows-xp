# 统一高考移民人数为18人

**类别**：A类
**优先级**：高
**依赖**：无
**修改文件**：
- `CLAUDE.md`
- `docs/设计.md`（禁用词列表部分）
- `docs/大纲.md`（阶段二隐藏列表部分）

## 背景

各文档对高考移民人数的描述严重不一致：CLAUDE.md 和设计.md 禁用词列表残留"15人"，大纲.md 阶段二隐藏列表出现"15人"。主体叙事（大纲.md 操作流程段、解密.md 新闻报道段、设计.md 信息分阶段释放表）已统一为"18人"。残留的"15人"会误导开发者实现错误的禁用词过滤逻辑。

## 具体修改内容

1. `CLAUDE.md`：将所有"15人"替换为"18人"
2. `docs/设计.md` 禁用词列表：将"15人"替换为"18人"
3. `docs/大纲.md` 阶段二隐藏列表：将"15人"替换为"18人"

不修改已经正确使用"18人"的段落。

## 验收标准

- 在 CLAUDE.md、设计.md、大纲.md 中全局搜索"15人"，结果为零
- 所有文档中高考移民人数统一为"18人"
- 设计.md 禁用词列表中的人数条目为"18人"
