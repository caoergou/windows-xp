# 创建短信记录断章取义版

**类别**：B类
**优先级**：高
**依赖**：无
**修改文件**：`src/data/documents/police_record_copy.txt`（修改现有文件，将短信截图内容替换为断章取义版）

## 背景

短信记录完全缺失，是当前最具阻断性的内容缺口之一。断章取义版短信是阶段二红鲱鱼系统的核心误导证据，缺失会导致"陈默=凶手"的误导链缺少动机环节，推理型玩家会因为没有动机而对误导产生怀疑。

## 具体修改内容

在 police_record_copy.txt 中，在现有内容的适当位置（警方现场勘查记录之后）插入以下短信截图内容，标注"警方存档副本——相关通讯记录摘录"：

```
2016-02-14 22:37  陈默->赵威："明天下午他会去烂尾医院那边。"
2016-02-14 22:41  赵威->陈默："知道了。"
2016-02-14 22:43  陈默->赵威："书包里有东西。你知道该怎么做。"
2016-02-14 22:45  赵威->陈默："放心。"
```

关键设计要求：
- 这个版本看起来完整、自洽，没有明显的截断痕迹
- 不标注"节选"或"部分"等字样，让玩家认为这就是全部
- 标注来源为"警方存档副本"，暗示这是官方记录（为阶段三"警方信息筛选本身就是掩盖"的揭露埋下伏笔）

## 验收标准

- police_record_copy.txt 中包含上述四条短信内容
- 短信内容标注为"警方存档副本"
- 内容看起来完整，没有截断标记
- 玩家读完后可以推断：陈默告知赵威林晓宇的行踪，并暗示赵威"知道该怎么做"
