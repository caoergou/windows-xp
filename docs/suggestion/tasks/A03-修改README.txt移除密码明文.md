# 修改 README.txt 移除第二层密码明文

**类别**：A类
**优先级**：高
**依赖**：无
**修改文件**：`src/data/documents/README.txt`（或 filesystem.json 中对应的文件内容字段）

## 背景

当前 README.txt 直接给出了父亲文件夹第二层密码 20151123，使谜题5（父亲第二层）完全没有推理过程。玩家打开 README 就能看到两层密码，破坏了解密体验。第二层密码应该通过阅读工作日志推理得出。

## 具体修改内容

将 README.txt 中直接给出第二层密码的内容，替换为以下暗示性表述：

```
小灯，

如果你看到这个文件夹，说明我已经走了。

第一层的密码你知道，是你的生日。
第二层的密码，是爸爸的灯熄灭的那一天。
你需要读完爸爸的日志，才能找到那一天。

爸爸
```

同时在文件末尾增加一句话：
```
爸爸对不起你，也对不起那个孩子。
```

"那个孩子"在阶段一读来模糊，在阶段四读来指向林晓宇，形成跨阶段伏笔。

## 验收标准

- README.txt 中不再出现"20151123"或任何明确的第二层密码数字
- 文件中包含"灯熄灭的那一天"的暗示性表述
- 文件中包含"对不起那个孩子"的伏笔句
- 玩家需要阅读工作日志2015.11.23条目才能推理出密码
