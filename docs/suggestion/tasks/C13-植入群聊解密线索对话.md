# task-02：群聊线索植入

谜题编号：前置线索（服务于谜题1-4）
难度：无（被动阅读）
所属阶段：阶段一（玩家在此阶段浏览群聊，线索自然植入）
预计玩家用时：15-20分钟（浏览全部群聊）

---

## 说明

群聊"山顶事务所"是阶段一的核心内容。玩家登录QQ后，浏览群聊记录，在不知情的情况下接触到所有谜题的前置线索。以下是需要植入的全部线索对话，含完整文本和植入位置说明。

---

## 对话1：EXIF科普（2015.11.05）

服务谜题：谜题1（加密相册）

```
林晓宇：今天学到一个新技能！原来照片里藏着很多信息。
陈默：什么信息？
林晓宇：叫EXIF数据。在Windows XP里，右键点击照片，选择"属性"→"摘要"→"高级"，就能看到拍摄时间、相机型号、光圈快门这些。
夏灯：哇，还有这种操作！
林晓宇：对啊，这样就能知道一张照片是什么时候、用什么设备拍的。我爸以前说，照片会说话，现在我懂了。
```

植入位置：群聊时间线中段，距离"光之公式"对话约20条消息之前。

---

## 对话2：光之公式（2015.11.20，含拼接规则）

服务谜题：谜题1（加密相册）、谜题3（加密日志，"光圈是眼睛本身"）

```
林晓宇：今天终于找到我最喜欢的参数组合了！
陈默：什么参数？
林晓宇：f/2.8（光圈大，能看清人），1/125（快门快，能定格瞬间），ISO400（感光度低，保持真实）。我把这三个参数叫做"光之公式"。
夏灯：好诗意的名字！
林晓宇：哈哈，这是我的信仰。我把这三个参数的纯数字部分拼在一起，当成我的专属密码。28、125、400——光之公式的数字版。
陈默：你这密码也太好猜了吧
林晓宇：只有懂我的人才能猜到。
```

植入位置：群聊时间线中段，紧跟EXIF科普对话之后约20条消息处。

拼接规则说明（供开发参考）：
- f/2.8 → 纯数字部分 → 28
- 1/125 → 纯数字部分 → 125
- ISO400 → 纯数字部分 → 400
- 拼接结果：`28125400`

## 对话3：光圈是眼睛本身（2015年11月前后）

服务谜题：谜题3（加密日志，密码提示"光圈是眼睛本身"）

```
陈默：你那三个参数里，你最喜欢哪个？
林晓宇：f/2.8。光圈是最重要的，决定你能看到多少。
陈默：快门和ISO呢？
林晓宇：快门是时间，ISO是代价。但光圈是眼睛本身。
```

植入位置：群聊时间线中段，与"光之公式"对话相邻（前后均可）。

注意：这段对话是谜题3相册描述末尾"光圈是眼睛本身"的来源，玩家在阶段三解锁相册后看到这句话，需要能回忆起这段群聊。

## 对话4：xiaodeng加油（日期：2015年某日）

服务谜题：谜题2（父亲文件夹第一层，密码 `xiaodeng`）

```
夏灯：爸爸今天在我日记本上写了"xiaodeng加油"，他连拼音都不会打，哈哈
林晓宇：哈哈哈，你爸太可爱了
陈默：这也太萌了
夏灯：他说这是"国际标准拼音"，我笑死了
林晓宇：你爸爸真的很爱你
夏灯：……嗯
陈默：你爸爸是真的很爱你，夏灯
夏灯：我知道的
```

植入位置：群聊时间线靠前段，让玩家在阶段一就看到"xiaodeng"这个字符串，但当时不知道它是密码。

设计要点：这段对话有8条消息，形成一个自然的话题停顿，玩家容易停留阅读。"xiaodeng"以引号形式出现，视觉上显眼，便于记忆。

## 对话5：藏头诗提示（《白夜行》那段，2016年1月前后）

服务谜题：谜题4（日志内的隐藏信息，首字藏头规则）

```
林晓宇：最近在看《白夜行》，里面有人把信息藏在诗里
陈默：怎么藏？
林晓宇：最简单的方式——每行第一个字连起来读
夏灯：哇，这也太聪明了
林晓宇：我也试着写了一首，发在日志里了，加密的
陈默：等我破解你的密码再去看
林晓宇：哈哈，你猜猜看，首字连读，七个字
夏灯：好神秘，我也想看
林晓宇：等你们破解了再说
```

植入位置：群聊时间线靠后段（2016年1月前后），距离谜题4触发（阶段三）约100分钟游戏时间。

设计要点：
- 这段对话有9条消息，形成一个完整的话题，玩家容易停留阅读
- 林晓宇明确说出"每行第一个字连起来读"和"七个字"，是谜题4的核心规则
- "加密的"暗示玩家需要先解锁日志才能看到这首诗
- 这段对话在阶段一就告诉玩家规则，但当时日志还没解锁，玩家只是记住了这个信息

## 群聊整体结构说明

群聊时间跨度：2015年9月（开学）至2016年2月14日（林晓宇坠楼前一天）

内容分布建议：
- 2015年9-10月：日常对话，建立三人关系，轻松愉快
- 2015年11月：EXIF科普、光之公式、光圈是眼睛本身（谜题线索密集植入）
- 2015年12月：气氛开始变化，林晓宇话少了，陈默偶尔问他还好吗
- 2016年1月：藏头诗提示对话，林晓宇说"我写了一首诗"
- 2016年2月1-14日：最后几条消息，气氛沉重，最后一条是2月14日

最后一条消息（2016.02.14）：

```
林晓宇：今天图书馆见到陈默了。
林晓宇：我们谈了很久。
林晓宇：我想通了一些事。
```

（之后再无消息。）
