# 补充群聊记录至150-200条并增加悬疑钩子

**类别**：B类
**优先级**：高
**依赖**：无
**修改文件**：`src/data/qq/chats/mountain_office.json`（或对应的群聊数据文件）

## 背景

群聊记录当前129条，设计要求150-200条。缺口主要在2014年9月至2015年10月的友谊深化期。同时，阶段一60分钟纯浏览缺乏悬疑钩子，玩家可能在大量日常对话中迷失方向。

## 具体修改内容

在现有129条基础上，补充约30-50条消息，重点补充以下内容：

**友谊深化期（2014年9月-2015年8月，约15-20条）**：
- 山顶事务所成立纪念对话（2014年9月15日）
- 林晓宇第一次提到父亲的相机（2014年9月5日）："今天翻出了爸爸的相机。Canon EOS 550D，背带上刻着'Camera is the 3rd eye'。"
- 陈默提到奶奶高血压开始定期检查（2014年11月）

**国家专项计划讨论（2015年8月，约5-8条）**：
- 林晓宇和陈默讨论国家专项计划（2015年8月16日）："好的政策也可能被用来做坏事。"
- 陈默问"是不是因为张雨学姐"，林晓宇说"我是在说一件不公平的事"（2015年8月20日）

**中期悬疑钩子（2015年10-11月，约5-8条）**：
- 陈默提到学校系统漏洞（2015年10月18日）："这个学校的系统真的很烂，随便就能进去。"
- 在群聊记录约30分钟阅读处（2015年10-11月间）插入一条撤回消息：
  ```
  2015-11-05 22:33
  陈默撤回了一条消息
  ```
  此后陈默语气明显变化，变得更加谨慎和简短。

**阶段一结尾钩子（2016年2月14日晚，必须包含）**：
```
2016-02-14 21:15
林晓宇撤回了一条消息

2016-02-14 21:16
林晓宇："如果我出事了，记得去看我的相册。密码是我的信仰。"

2016-02-14 21:17
[夏灯在线，未回复]
[陈默在线，未回复]

2016-02-14 21:18
林晓宇："算了，没事。"
```

## 验收标准

- 群聊消息总数达到150条以上
- 包含2014年9月5日林晓宇提到父亲相机的消息
- 包含2015年8月国家专项计划讨论（两段对话）
- 包含2015年10月18日陈默提到系统漏洞的消息
- 包含2015年11月陈默撤回消息的悬疑钩子
- 包含2016年2月14日晚林晓宇"如果我出事了"的阶段一结尾钩子
- 张雨年级已修正为高三（与 A07 任务一致）
