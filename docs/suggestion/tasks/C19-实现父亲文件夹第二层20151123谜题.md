## 解锁内容：《给夏灯.txt》结构大纲

约8000字，分四部分。以下为结构说明，供写作参考，不是全文。

### 开篇：为什么叫你"灯"（约1000字）

```
小灯，当年给你起这个名字，是希望你能成为照亮黑暗的那束光。
我当老师三十年，一直相信教育能改变命运，相信知识能让寒门子弟走出大山。
但这十年，我看到的是另一个世界。
```

核心内容：
- 夏建国的教育理想，三十年的坚守
- "灯"这个名字的来历和寄托
- 为什么他要写这封信

### 第一部分：我做了什么（约2000字）

核心内容：
- 2015年11月23日，校长拿来一份"就读证明"，让他签字
- 他签了。因为他知道不签会怎样——调离岗位、女儿的前途
- 之后的两年，他又签了十几份
- 他看到陈默深夜在机房，他猜到了什么，但他装作不知道
- 他看到林晓宇来找他，眼里有光，问他"被迫犯罪算犯罪吗"
- 他告诉林晓宇"不要声张，这件事很复杂"
- 三个月后，林晓宇死了

关键坦白（与谜题5回收站文件形成呼应）：

```
林晓宇死后第二天，我在他的电脑上发现了一个截图文件。
我知道它意味着什么。
2月16日上午，我把它删了。
我告诉自己这是为了保护你，但我知道，这只是借口。
```

### 第二部分：我听到的、推测的（约2000字）

核心内容：
- 教育局副局长的暗示："这些学生的学籍，你就别管了"
- 校长的话："老夏，你有女儿，我有儿子，咱们都是为了孩子"
- 高考移民产业链的运作（基于观察的推测）：每年约6人，持续多年，保护伞网络
- 那些被挤占名额的孩子：张雨（589分）、周磊（582分）、方欣

### 第三部分：我的教育理想是怎么死的（约2000字）

核心内容：

```
有一张看不见的网。网的一边是权力和金钱，另一边是我们这些普通人。
你敢挣扎，网就收紧。你不挣扎，就会慢慢窒息。
我选择了不挣扎。我告诉自己"这是为了你"，但我知道，这只是借口。
```

- 2016年2月20日的对话：校长说"老夏，你有女儿，我有儿子，陈默有奶奶，我们都有软肋"
- 夏建国对自己软弱的剖析，不为自己辩护，只是陈述

### 第四部分：我对你的希望（约3000字）

核心内容：

```
小灯，我把这些都留给你，不是为了让你原谅我。
你的名字是"灯"，我希望你能照亮我不敢照亮的黑暗。
如果你看到这封信，说明我已经走了。我很抱歉把这个重担留给你。
但我相信，你会比我勇敢。
```

- 夏建国对女儿的最后嘱托
- 他知道自己没有资格要求原谅
- 他把希望寄托在夏灯身上，但同时也在问：这样做对吗？

## 情感反应

玩家读完《给夏灯》，理解了夏建国的全部——他的软弱、他的愧疚、他对女儿的希望。

特别是"我把它删了"这句话，与谜题5中回收站文件的发现形成呼应：玩家在阶段三发现了那个截图，现在才知道是父亲删除的。这个呼应让阶段四的情感冲击更深。

调查笔记最终弹出，夏灯写下：

```
爸爸。
我不知道该原谅你，还是该恨你。
但我知道，我会继续。
```
