# 补充夏父工作日志中张雨前置铺垫

**类别**：C类
**优先级**：中
**依赖**：无
**修改文件**：`src/data/documents/`（父亲工作日志第一层，具体文件路径取决于 filesystem.json 中的日志文件组织方式）

## 背景

张雨的完整故事要到阶段四才通过调查报道呈现，缺乏前期铺垫。在阶段二的父亲日志第一层中增加一段关于张雨的简短记录，可以让张雨的形象在阶段二就开始建立，到阶段四完整呈现时情感冲击更强。

## 具体修改内容

在父亲工作日志2015年8月的条目中，添加以下记录：

```
2015年8月15日

今天在照相馆门口看到张雨和她妈妈。
张雨考了589分，国家专项没通过。
她妈妈是县医院清洁工。
我当时不明白为什么，现在我明白了。
```

这段记录在阶段二读来只是夏父的一个模糊的疑惑，玩家不会立刻联想到高考移民。在阶段四，玩家会意识到：夏父当时已经注意到了异常，但选择了"不明白"（实际上是不敢明白）。

同时，在2015年11月23日（第一次签字）的日志条目中，可以加入一句：

```
我想到了张雨。然后我签了字。
```

这句话在阶段二读来意义模糊，在阶段四读来是夏父最深的愧疚之一。

## 验收标准

- 父亲工作日志2015年8月条目包含张雨589分的简短记录
- 记录包含"她妈妈是县医院清洁工"的细节
- 2015年11月23日条目包含"我想到了张雨。然后我签了字。"
- 两处记录在阶段二读来意义模糊，在阶段四读来有完整含义
