# 统一涉案金额为2000万

**类别**：A类
**优先级**：高
**依赖**：无
**修改文件**：
- `docs/大纲.md`（阶段二隐藏列表部分）
- `docs/设计.md`（禁用词列表部分）

## 背景

涉案金额在各文档中出现三个不同数字：大纲.md 阶段二隐藏列表和设计.md 禁用词列表中出现"250万"，大纲.md 操作流程段出现"1800-2000万"，解密.md 和设计.md 记者手记统一为"2000万"。"250万"完全无法解释来源，是早期版本残留，会导致禁用词过滤逻辑错误。

## 具体修改内容

1. `docs/大纲.md` 阶段二隐藏列表：将"250万"替换为"2000万"
2. `docs/设计.md` 禁用词列表：将"250万"替换为"2000万"
3. `docs/大纲.md` 操作流程段：将"1800-2000万"统一为"2000万"（与解密.md 和设计.md 记者手记保持一致）

## 验收标准

- 在大纲.md 和设计.md 中全局搜索"250万"，结果为零
- 在大纲.md 中全局搜索"1800"，结果为零
- 所有文档中涉案金额统一为"2000万"
