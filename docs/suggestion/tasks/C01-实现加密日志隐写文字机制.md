# 实现加密日志隐写文字机制

**类别**：C类
**优先级**：高
**依赖**：依赖 C16（C16 确认了可见文字和隐藏密码的最终内容，本任务负责实现机制）
**修改文件**：
- `src/apps/QZone.jsx`
- `src/data/qzone/linxiaoyu/blog.json`（或 encrypted_diary.json）

## 背景

谜题3（加密日志《第三只眼》）的核心解密路径是：玩家在林晓宇 QQ 个人说明区域选中白色隐藏文字，发现密码 `camera3rdeye`，直接输入解锁日志。当前代码中 QZone.jsx 直接以纯文本渲染个人说明，无隐写机制。这是整个解密系统中最严重的阻断，也是游戏核心主题（"看不见的才是真相"）的具象化表达，必须实现。

## 具体修改内容

**数据层修改（index.json 或 blog.json）**：

在林晓宇 QQ 空间用户数据中，个人说明字段改为：
```json
{
  "signature": {
    "visible": "相机是我的第三只眼。",
    "hidden": "camera3rdeye"
  }
}
```

**代码层修改（QZone.jsx）**：

在个人说明渲染处，实现以下机制：
1. 渲染 signature.visible 文字（正常可见）
2. 在 visible 文字之后插入 signature.hidden 文字，设置为白色（color: #ffffff），背景与页面一致
3. 支持鼠标拖选或 Ctrl+A 时，隐藏文字变为可见
4. 视觉效果：隐藏文字被选中时从白色变为深色，模拟暗房照片显影效果

最终渲染效果（未选中时）：
```
相机是我的第三只眼。
（其后有不可见的白色文字"camera3rdeye"）
```

选中后：
```
相机是我的第三只眼。camera3rdeye
```

HTML 实现参考：
```html
<div class="qq-signature">
  <span style="color: #000;">相机是我的第三只眼。</span>
  <span style="color: #ffffff; background: #ffffff; user-select: text;">camera3rdeye</span>
</div>
```

## 验收标准

- 林晓宇 QQ 空间个人说明可见文字为"相机是我的第三只眼。"
- 可见文字之后有白色隐藏文字 `camera3rdeye`，未选中时不可见
- 用鼠标拖选或 Ctrl+A 选中文字后，隐藏文字变为可见
- 玩家直接看到 `camera3rdeye`，无需任何转换推导，可直接输入密码框
- 解锁后显示完整日志内容（现有逻辑保持不变）
