# 修正群聊中张雨年级描述为高三

**类别**：A类
**优先级**：中
**依赖**：无
**修改文件**：`src/data/qq/chats/mountain_office.json`（或对应的群聊数据文件）

## 背景

群聊数据中2014年12月20日林晓宇说"今天在图书馆看到高二的张雨学姐"，但根据时间线.md，张雨在2015年6月参加高考。如果2014年12月张雨是高二，则2015年6月她应该是高二下学期，不应该参加高考。张雨在2014年12月应该是高三才符合时间线逻辑。

## 具体修改内容

在群聊数据文件中，找到2014年12月20日林晓宇的消息，将"高二的张雨学姐"修改为"高三的张雨学姐"。

## 验收标准

- 群聊数据中2014年12月20日的消息显示"高三的张雨学姐"
- 修改后张雨的年级与时间线.md中2015年6月参加高考的记录逻辑一致
