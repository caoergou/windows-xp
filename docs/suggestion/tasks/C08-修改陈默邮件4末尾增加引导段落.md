# 修改陈默邮件4末尾增加引导段落

**类别**：C类
**优先级**：高
**依赖**：无
**修改文件**：`src/data/email/inbox/email_chenmo_004.txt`

## 背景

阶段二结尾钩子需要陈默在承认"在现场、眼镜是我的"之后，主动引导玩家去看林晓宇的日志。一个"凶手"主动让受害者的日志来评判自己——这个反常行为会让玩家产生疑惑，驱动他们迫切地想要破解日志，同时也为阶段三的反转埋下伏笔。

## 具体修改内容

在 email_chenmo_004.txt 的现有内容末尾，添加以下段落：

```
我知道你现在怎么想我。
我不打算辩解。

但在你做任何决定之前，我希望你先去看林晓宇的日志。
他在日志里写了一些东西。
我不知道他有没有写到我。
但我知道他比任何人都更了解那天发生了什么。

如果你看完日志之后还是认为我是凶手，
我接受。

陈默
```

## 验收标准

- email_chenmo_004.txt 末尾包含上述引导段落
- 段落中陈默没有为自己辩解，只是引导玩家去看日志
- "他比任何人都更了解那天发生了什么"暗示日志中有关于2月15日的记录
- 邮件以"我接受"结尾，语气克制，符合陈默的人物风格
