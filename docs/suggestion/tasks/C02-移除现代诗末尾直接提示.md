# 移除现代诗末尾直接提示

**类别**：C类
**优先级**：高
**依赖**：无
**修改文件**：`src/data/qzone/linxiaoyu/blog.json`（或 encrypted_diary.json）中的诗歌内容

## 背景

现代诗《第三只眼》末尾直接写了"（每行第三个字连起来读）"，将一个精妙的4星难度谜题降为1星填空题。群聊2015.12.28中林晓宇已经提到"每行第三个字连起来读"，玩家应该通过这条群聊线索自行联想，而不是被直接告知。

## 具体修改内容

在 encrypted_diary.json（或 blog.json）中，找到现代诗《第三只眼》的内容字段，删除末尾的"（每行第三个字连起来读）"提示文字。

诗歌本身的15行内容保持不变，只删除末尾的括号提示。

如果提示系统（puzzle_hints.json）中有对应的渐进提示，确保在玩家尝试多次后，提示系统会引导玩家"回去看看群聊记录，林晓宇曾经提到过一种读诗的方法"，而不是直接给出答案。

## 验收标准

- encrypted_diary.json（或 blog.json）中诗歌内容末尾不包含"（每行第三个字连起来读）"
- 诗歌15行内容完整保留
- puzzle_hints.json 中对应谜题的提示内容指向群聊线索，而非直接给出方法
