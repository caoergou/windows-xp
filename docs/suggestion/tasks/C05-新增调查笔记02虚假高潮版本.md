## 背景

阶段二末尾需要一个"虚假高潮"——玩家通过夏灯的推理感到"案件已经基本清晰"，但最后两行留下一个无法解释的疑点，驱动玩家进入阶段三。当前调查笔记-02的内容偏向分析性，缺少情感震荡和关键疑点。

## 具体修改内容

将调查笔记-02的内容修改为以下文本（在玩家看完父亲日志第一层、相册照片、陈默邮件4之后自动触发）：

```
调查笔记-02

我现在知道的：

2015年11月23日，陈默开始篡改学籍数据。
2015年12月2日凌晨，陈默在机房完成数据导入。
林晓宇发现了陈默的操作，开始调查。
陈默警告林晓宇"会被格式化"。
2016年2月14日，两人在图书馆对峙，陈默哭了。
2016年2月14日晚，陈默发短信给赵威，告知林晓宇的行踪。
2016年2月15日，林晓宇坠楼，陈默在现场，眼镜掉落。

结论：
陈默为了保护自己（篡改数据的秘密），
通知赵威对林晓宇动手。
他在现场，可能亲眼目睹，可能参与。

但我还有一个问题：
如果陈默是凶手，为什么他要主动联系我？
为什么他要让我去看林晓宇的日志？

也许他以为日志里没有对他不利的内容。
也许他在赌。

我需要看那个日志。
```

## 验收标准

- 调查笔记-02包含上述完整内容
- 笔记在玩家看完相册照片和陈默邮件4后自动触发（打字机效果）
- 笔记结尾留下"为什么他要让我去看日志"的疑点
- 疑点驱动玩家进入阶段三（破解加密日志）
- 笔记语气符合夏灯的视角（第一人称，情感化）
