# 提议：引导模式交互系统

## 核心机制

### 1. 阶段引导对话框

**触发时机**：每个阶段开始时和关键节点完成后

**视觉效果**：

- 屏幕全部变暗（遮罩层）
- 中央出现 XP 风格对话框
- 显示当前阶段标题、副标题和目标列表
- 提供初始提示

**功能**：

- 明确告知玩家当前阶段和调查目标
- 在任务栏添加"调查进度"图标，随时查看目标完成情况
- 自动检测目标完成，触发阶段总结

### 2. 陈默邮件引导系统

**核心流程**：

1. 陈默发送引导邮件（引导型/验证型/揭示型）
2. 邮件中包含"立即回复"按钮
3. 点击后进入引导模式对话框
4. 玩家从多个选项中选择回答
5. 系统自动生成回复内容发送给陈默
6. 陈默快速回复（5秒内）：
   - 选择正确 → 鼓励并引导下一步
   - 选择错误 → 温和纠正并提供线索

**邮件时机控制**：

- 关键节点触发（完成目标后）
- 玩家卡关超过10分钟时触发
- 任务栏显示新邮件通知，不强制立即查看

### 3. 多账号调查

**机制**：

- 在父亲文件夹中发现"林晓宇QQ密码.txt"
- 支持快速切换账号（任务栏按钮）
- 林晓宇视角包含：草稿箱、私密说说、威胁邮件、撤回消息等半隐藏信息
- 双视角对比，拼凑完整真相

## 设计原则

- 引导可选：玩家可关闭引导模式
- 渐进式提示：从温和到具体到直接答案
- 保持沉浸感：XP风格设计、打字机效果、音效配合
- 平衡探索自由：不过度打断玩家节奏
