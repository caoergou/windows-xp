# 《山月无声》解密设计

*社会派推理的渐进式揭露体验*

---

## 核心设计理念

### 设计原则

**原则1：信息源合理性**

每个角色只能提供他能获取的信息：

- **林晓宇**：碎片化情绪表达，文学化暗示，证据照片（不是详细的侦探记录）
- **夏父**：他的观察记录、教务系统日志、罪行忏悔（不是私人对话内容）
- **陈默**：完整证据通过自首后的媒体报道公开（不通过邮件喂剧情）

**原则2：数字考古体验**

```
公开层（60%）：QQ空间说说、群聊记录、贴吧帖子、照片
      ↓ 玩家直接浏览，形成初步认知

半公开层（25%）：QQ空间加密相册、加密日志
      ↓ 简单解密，获得关键证据

私密层（15%）：夏父忏悔日记、二次加密文件夹
      ↓ 复杂解密，揭示夏父罪行和指向陈默
```

**原则3：红鲱鱼优先级**

```
简单铺垫（20分钟）：情绪自杀假象
            ↓
核心误导（90分钟）：陈默是凶手
            ↓
震撼反转（30分钟）：陈默也是受害者
```

## 游戏时长与阶段划分

**总时长**：180分钟（3小时）

- 解密时长：约60分钟
- 阅读+浏览时长：约120分钟

### 四阶段结构

| 阶段 | 时长 | 核心问题 | 情感状态 | 认知层次 |
|-----|------|---------|---------|---------|
| **第一阶段**<br>个体悲剧 | 60分钟 | 林晓宇是怎么死的？<br>他在调查什么？ | 怀旧、困惑 | 意外坠楼→林发现了什么？ |
| **第二阶段**<br>朋友还是凶手 | 50分钟 | 陈默是不是参与了？<br>2月15日发生了什么？ | 愤怒、背叛 | 陈默至少是共犯 |
| **第三阶段**<br>真相反转 | 40分钟 | 陈默到底是不是凶手？<br>如果不是，真相是什么？ | 震撼、理解 | 陈默也是受害者 |
| **第四阶段**<br>系统揭露 | 30分钟 | 这背后是什么？ | 沉重、反思 | 系统性腐败 |

## 红鲱鱼系统

### 红鲱鱼#1：情绪自杀（简单铺垫，20分钟）

**误导证据**：
- QQ空间说说越来越阴郁
- 引用《白夜行》等抑郁文学
- 贴吧"偷拍狂"网络暴力
- 警方结论"心理压力过大"

**作用**：
- 让玩家前20分钟以为是"网络暴力导致自杀"
- 为后续反转做铺垫

### 红鲱鱼#2：陈默=凶手（核心误导，90分钟）

**误导证据链**（6条）：
1. 夏父日志：陈默在机房做过异常操作（11.23、12.02凌晨）
2. 相册照片：12.02凌晨机房有人
3. 夏父日志：2.14林晓宇和陈默吵架，陈默哭了，第二天林晓宇就死了
4. 警方记录：现场照片有黑框眼镜，但报告强行掩盖（引起玩家对警方的不信任，但眼镜本身指向陈默）
5. 警方记录：证人看到"戴眼镜的男生"
6. 陈默邮件：承认在现场、眼镜是他的

**推翻证据**（4条）：
1. 林晓宇日志：陈默是被迫的，林晓宇理解他
2. 林晓宇日志：陈默想陪林晓宇去，想保护他
3. 新闻报道公开完整短信记录：陈默警告赵威别去
4. 陈默完整证词：去现场是为了救林晓宇

**情感冲击**：从愤怒、背叛到理解、同情

### 红鲱鱼#3：林晓宇的秘密恋情（轻度误导）

**误导线索**：
- 群聊和空间说说中反复提到"在图书馆看到她"
- "589分，她一定会去很好的大学"

**作用**：
- 增加人物立体感
- 实际上是张雨学姐，也是调查的起点
- 为"偷拍狂"谣言提供基础

## 提示系统设计

### 核心谜题（谜题3, 6, 7）

渐进式提示，提高触发门槛：

```
错误5次  → 方向性提示（"试试在群聊中搜索..."）
错误8次  → 具体领域提示（"林晓宇提到过他的座右铭"）
错误12次 → 详细线索位置（"搜索'第三只眼'"）
错误15次 → 半答案（"camera****"）
错误20次 → 完整答案
```

### 辅助谜题（谜题2, 4, 5）

更快给出提示：

```
错误3次  → 方向性提示
错误5次  → 详细提示
错误10次 → 提供跳过选项
```

### 陈默邮件提示（可选）

当玩家卡住超过10分钟，陈默可发送提示邮件，提供方向性引导。
