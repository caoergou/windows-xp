# 《山月无声》解密设计

*社会派推理的渐进式揭露体验*

---

## 核心设计理念

### 设计原则

**原则1：信息源合理性**

每个角色只能提供他能获取的信息：

- **林晓宇**：碎片化情绪表达，文学化暗示，证据照片（不是详细的侦探记录）
- **夏父**：他的观察记录、教务系统日志、罪行忏悔（不是私人对话内容）
- **陈默**：完整证据通过自首后的媒体报道公开（不通过邮件喂剧情）

**原则2：数字考古体验**

```
公开层（60%）：QQ空间说说、群聊记录、贴吧帖子、照片
      ↓ 玩家直接浏览，形成初步认知

半公开层（25%）：QQ空间加密相册、加密日志
      ↓ 简单解密，获得关键证据

私密层（15%）：夏父忏悔日记、二次加密文件夹
      ↓ 复杂解密，揭示夏父罪行和指向陈默
```

**原则3：红鲱鱼优先级**

```
简单铺垫（20分钟）：情绪自杀假象
            ↓
核心误导（90分钟）：陈默是凶手
            ↓
震撼反转（30分钟）：陈默也是受害者
```

## 游戏时长与阶段划分

**总时长**：180分钟（3小时）

- 解密时长：约60分钟
- 阅读+浏览时长：约120分钟

### 四阶段结构

| 阶段 | 时长 | 核心问题 | 情感状态 | 认知层次 |
|-----|------|---------|---------|---------|
| **第一阶段**<br>个体悲剧 | 60分钟 | 林晓宇是怎么死的？<br>他在调查什么？ | 怀旧、困惑 | 意外坠楼→林发现了什么？ |
| **第二阶段**<br>朋友还是凶手 | 50分钟 | 陈默是不是参与了？<br>2月15日发生了什么？ | 愤怒、背叛 | 陈默至少是共犯 |
| **第三阶段**<br>真相反转 | 40分钟 | 陈默到底是不是凶手？<br>如果不是，真相是什么？ | 震撼、理解 | 陈默也是受害者 |
| **第四阶段**<br>系统揭露 | 30分钟 | 这背后是什么？ | 沉重、反思 | 系统性腐败 |

## 提示系统设计

### 核心谜题（谜题3：加密日志，谜题4：现代诗）

渐进式提示，提高触发门槛：

```
错误5次  → 方向性提示（"试试在群聊中搜索林晓宇关于诗的讨论"）
错误8次  → 具体领域提示（"林晓宇提到过他的个人说明里有什么"）
错误12次 → 详细线索位置（"QQ个人说明里可能有隐藏内容，试着选中文字"）
错误15次 → 半答案（"camera****" 或 "每行第3个字"）
错误20次 → 完整答案
```

### 辅助谜题（谜题1：相册，谜题2：父亲第一层，谜题5：父亲第二层）

更快给出提示：

```
错误3次  → 方向性提示
错误5次  → 详细提示（如"群聊里的相机参数"、"你的生日"、"2015年11月23日"）
错误10次 → 提供跳过选项
```

### 陈默邮件提示（可选）

当玩家卡住超过10分钟，陈默可发送提示邮件，提供方向性引导。
