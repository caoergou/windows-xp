# QQ-CLASSIC-UI.md — 千禧年代经典 QQ 界面考据（issue #119 研究文档）

> Research dossier for **#119 (QQ Messenger MVP)**: a pixel-level reference of the
> classic Tencent QQ client (2003–2007) — main panel / buddy list, chat window,
> login flow, emoticons, sounds and tray behaviors — plus how each finding maps
> onto this engine's existing assets. Written in Chinese because every primary
> source (and the culture pack itself) is Chinese; hex values, sizes and file
> anchors are language-neutral.
>
> 证据分级：**[确凿]** = 有真实截图 / 原版安装包提取素材 / 多来源一致；
> **[推测]** = 记忆或间接证据，落地前建议再对照一张真图。
> 黄金参照物：`mengkunsoft/QQ2006`（GitHub 97★ 像素级网页复刻，**素材全部提取自
> 原版 QQ2006 安装包**，作者许可自由分享修改、保留出处），其 preview.png 一图含
> 登录框 / 登录中 / 聊天窗 / 主面板四个界面，本文所有"已核对"的色值和尺寸均逐行
> 读取其 CSS 并对截图放大裁剪验证。

## 0. 版本基准：以 QQ2006 为视觉蓝本

| 版本 | 发布 | 关键界面变化 |
|---|---|---|
| QQ2003 | 2003 | 首次大改版，"天蓝色界面"定下其后十年的主面板框架；开始支持换肤 |
| QQ2004/2004II | 2004 | 主面板左侧新增窄竖栏（面板切换图标列）；16 级以上可自定义头像 |
| QQ2005 | 2005 | 深蓝主色、立体感增强；**好友列表改为头像靠左单列**；**首次引入个性签名**；XP 风格菜单 |
| **QQ2006** | 2006.12 正式版 | 界面更干净，信息分组清晰，**"经典 QQ 界面"就此定型**；主题包/换肤 |
| QQ2007/2007II | 2007 | 在线状态扩为 7 种；登录前 QQ 医生检测；**2007II Beta1 才引入窗口抖动**（仿 MSN Nudge） |

**结论**：#119 的 ARG 背景设定是 2005–2007，QQ2006 是三个理由下的最优基准——
① 它是"经典 QQ"的定型版；② 存在像素级参照物（见上）；③ 现有 `QQLogin.tsx`
标着 "QQ 2007"，但界面是凭空发挥的（见 §4 差距清单），无论叫 2006 还是 2007
都需要按真实结构重做。**建议：视觉按 QQ2006 复刻，版本文案可保留 2007**（两代
登录框结构相同，观感差异小于结构差异）。⚠️ 一个硬性年代约束：**窗口抖动是
QQ2007II（2007 年中）才有的功能**——若场景背景设在 2005/2006，抖动不该出现；
建议做成可配置彩蛋而非默认功能。

## 1. 主面板（好友列表窗口）[确凿，像素级核对]

窄长竖条窗口 **190×580px**，圆角 8px，全窗自绘皮肤（不是 XP 系统窗框）。
自上而下：

```
┌────────────────────────────┐
│ ① 标题栏 24px  蓝渐变 + 右侧 16×16 自绘按钮:      │
│    最小化 / 换肤(调色板) / 关闭                    │
│ ② 个人信息横幅 44px:                              │
│    32×32 自己头像(8,6) + 状态下拉小按钮(11×36)     │
│    + 昵称"痴情梦丶(在线)" (#071E81 粗体 8pt)       │
│    + 下行小图标: 邮件(0) / 安全中心 / 我的钱包      │
│ ③ 主体区:                                         │
│    左缘竖直面板栏 29px 宽(27×37 按钮×10:           │
│      QQ好友/互动空间/无线乐园/企业好友/网络杂志/    │
│      自定义/音乐中心/网络硬盘/综合业务/面板管理)    │
│    右侧好友盒: 白底, 1px #2367C3 边框              │
│      顶部页签按钮"QQ好友"                          │
│      分组条(手风琴): 我的好友(19/26) 家人(5/13)     │
│        同学(28/37) 陌生人 黑名单                   │
│      好友条目(见下)                                │
│      底部页签按钮: 手机好友 / 群/校友录 / 最近联系人 │
│ ④ 底部工具栏 62px:                                │
│    上排 16×16 业务小图标×8(手机短信/QQ游戏/TT浏览器 │
│      /QQ空间/QQ音乐/QQLive/QQ宠物…)                │
│    下排:【菜单】按钮 65×24(绿色光泽+企鹅+"菜单")    │
│      + 信息管理器 + "查找"按钮(47px 放大镜)         │
└────────────────────────────┘
```

### 分组条
- 高约 22px（`padding: 5px 0 5px 20px`），左侧 16×16 **绿色圆形三角箭头**
  （▶ 收起 / ▼ 展开），组名 `#101C4A` 宋体 12px。
- **分组名后带在线计数 `(在线/总数)`**，如"我的好友(19/26)"。
  [确凿——复刻截图如此；一位研究员推测该计数是 2008+ 特性，与实物矛盾，以实物为准]
- 固定系统分组：**陌生人**、**黑名单**（不可改名，无计数）；常见自建组：
  朋友、家人、同学（新建组默认名"未命名"）。
- 交互为手风琴式展开/收起。

### 好友条目（2005+ 单列布局）
- 行结构：40×40 头像（`padding: 4px 0 4px 4px`）+ 右侧两行：
  昵称（`#000400`，宋体 12px）/ 个性签名（灰色小字，单行截断）+
  15×15 业务角标（彩铃/手机 QQ 等）。
- **会员（红名）**：昵称 `#F00`，签名 `#FF7D7B`。
- **离线/隐身（他人视角）**：整行 `filter: grayscale(100%)`，文字 `#A8A8A8`
  —— 头像变灰是经典 QQ 最强的状态语言。[确凿]
- "离开"状态：彩色头像右下叠加小时钟角标。[推测]
- 经典系统头像共 **105 个** 40×40 卡通形象（中国娃娃、兔子、动漫脸等，浅蓝底），
  是互联网文化符号（腾讯 ISUX 官方做过像素重制确认该清单）。

## 2. 聊天窗口 [确凿，像素级核对]

默认 **500×460px**，皮肤底色 `#5DB7FF`，圆角 8px，**自绘标题栏**（26px，
非 XP 窗框；标题按钮 17×16 三态图）。自上而下：

1. **标题栏**：黄色便签纸小图标 + **`与 <昵称> 聊天中`**（白色粗体 12px）。
2. **大工具条 36px**（图标+文字按钮）：`短信 | 视频 | 语音 | 传文件 | 3D秀 | 邀请`
   —— 文字 `rgb(7,30,129)` 粗体；hover 浅蓝底 `rgb(130,199,255)` + 边框
   `rgb(70,136,201)`。背景渐变 `#A1D7FF → #5DB7FF`。
3. **对方信息条**：渐变 `#FFF → #B6D5F5`，底边框 `#84A6C6`；内容
   `20×20 头像 + 昵称(QQ号)：个性签名`，文字 `rgb(7,30,129)`，可点击。
4. **消息历史区**：纯白。**没有气泡**（见下）。
5. **小工具条 20px**（18×18 纯图标，hover 浅蓝框）：
   `A字体 | 😀表情 | 魔法表情 ┃ 发送图片 | 屏幕捕捉 | 聊天场景 | 礼物 | 语音消息`
   （┃ = 1px `rgb(114,149,188)` 分隔线）。点 "A" 展开第二行字体条
   （粗体/斜体/下划线/颜色）。
6. **输入区**：无边框白底 textarea，约 70px 高。
7. **底部按钮行 20px**：左 `聊天记录(H)` `消息模式(T)`；右 `关闭(C)`
   `发送(S)` + 紧贴 `↓` 下拉（切换 **Enter 发送 / Ctrl+Enter 发送**）。
8. **右侧栏 140px**（QQ2005/2006 特有，1px `rgb(53,111,175)` 边框）：三段
   可折叠面板 `▲对方形象»` / `▲个人空间»` / `▲我的形象»`；形象区显示 QQ 秀
   全身像；个人空间区：`摘要：… / 日记：48条/169评论 / 相册：… / 收藏：…`，
   **数字红色**，底 `#F6F6F6` 文字 `#0F0657`。

### 消息展示格式（时代辨识度最高的细节）[确凿]

```
昵称  HH:MM:SS          ← 彩色小字一行（时间与昵称同色，间距 8px）
    消息正文             ← 黑色，左缩进 15px，行高 1.4
```

- **对方昵称行 = 深蓝 `#210F95`；自己 = 深绿 `#2D794B`**（已对预览图放大逐像素
  核对）。真实客户端观感约纯蓝/深绿系，直接用复刻值即可。
- 全窗口字体：**宋体（SimSun）12px（9pt）**，与 XP 中文环境一致。
- 头部行只有 `昵称 + 时:分:秒`；`昵称(QQ号)` 全形式只出现在顶部信息条、
  群聊消息头和聊天记录导出（D 盘 `聊天记录.txt` 的「甲」对「乙」说格式即
  消息管理器导出样式——已有文件与考据吻合）。
- 正文颜色/字体可被发送方自定义（字体工具条），接收方按对方设置渲染。

### 通用按钮样式（登录框/聊天窗/主面板共用）[确凿]

```css
font: 12px SimSun;
background: linear-gradient(to bottom, #FFFFFF, #9FD4FF);
border: 1px solid #00558E;  border-radius: 3px;
/* hover: → #B9E1FF;  active: #97C5EC → #D2F8FD (反向);
   focus: 内圈黄环 #FFE762/#FCB833 */
```

## 3. 表情体系 [部分确凿]

- 经典**黄脸表情 2003 年加入**，按数字编号存储（微笑=14.gif、呲牙=13.gif）。
  16×16，橙色描边、高饱和黄，与现代 emoji 观感差异显著。
- 经典面板顺序（沿用至今，可放心用）：微笑 撇嘴 色 发呆 得意 流泪 害羞 闭嘴
  睡 大哭 尴尬 发怒 调皮 呲牙 惊讶 难过 酷 冷汗 抓狂 吐 偷笑 可爱 白眼 傲慢
  饥饿 困 惊恐 流汗 憨笑 大兵 奋斗 咒骂 疑问 嘘 晕 折磨 衰 骷髅 敲打 再见 …
- **快捷码是拼音缩写**：`/wx`微笑 `/pz`撇嘴 `/dy`得意 `/ll`流泪 `/cy`呲牙
  `/dk`大哭 `/hx`害羞 `/fd`发呆 `/shui`睡 `/gg`尴尬……（`/微笑` 汉字全名是
  更晚版本的写法 [推测]）。
- 表情面板：点黄脸按钮弹出无标题栏浮层，**密集网格**（约 24px 单元格、1px
  网格线、hover 蓝框），QQ2005+ 带"默认/自定义表情"页签。
- ⚠️ **与现有代码的冲突**：`src/utils/emojiRenderer.ts` 用 `[微笑]` 方括号码 →
  现代 Unicode emoji，注释自述"基于 2015 年左右的 QQ 表情"。方括号码 + 彩色
  emoji 都不是千禧年代的形态。#119 要求"emojiRenderer 终于成为活代码"，建议
  升级而非照搬：**代码层支持 `/wx` 缩写码（输入）+ `[微笑]`（数据可读性）双写法；
  渲染层输出 16px 内联"经典黄脸"**——用自绘 SVG/像素图（同项目其它图标的做法）
  或退而求其次 emoji + `filter: saturate(1.4)` 缩至 16px。素材版权见 §7。

## 4. 登录窗口与登录流程 [确凿，像素级核对]

QQ2006 登录框是**真正的 XP Luna 系统窗**（与自绘皮肤的主面板/聊天窗不同！
这对本项目是好消息：可直接复用引擎现有 XP 窗框）：

1. 标题栏："QQ用户登录" + 企鹅小图标 + XP 最小化/最大化(禁用)/关闭。
2. **横幅 47px**：左侧企鹅 + 胶囊 "QQ®2006" 艺术字 + 标语
   **"网络新生活，体验新感受!"** + 右上角腾讯网 logo。
3. **表单区**（底色 `#ECF6FF`，内框渐变 `#E4F3FF→#F0F8FF`、1px `#2F74C5`，
   padding 18px，窗宽 325px）：
   - `QQ号码 ▼`（label 带小三角，可切 号码/手机/Email 登录）+ 组合框
     （占位 `<请在这儿输入QQ号码>`，右端 XP 下拉钮）+ `申请号码` 按钮（75×20）；
   - `QQ密码` + 密码框（1px `#7F9DB9`）+ 蓝链接 "忘了密码？"；
   - 复选框：☑ 自动登录 ☐ 隐身登录。
4. 按钮行：左 `高级设置 ↓`，右 `登录` `取消`。
5. **登录中窗口**：与主面板同形的窄长条（190×580）——QQ 大 logo、蓝色滚动
   进度条、"正在登录"、`取消登录`。**登录成功后此窗原地"变成"主面板**，
   这个形态延续是经典记忆点，值得复刻。

### 现有 `QQLogin.tsx` 与真实 QQ2006/2007 的差距清单

| 维度 | 现状 | 真实形态 |
|---|---|---|
| 窗框 | 自绘蓝渐变头 "QQ 2007" | XP 系统窗 + 47px 品牌横幅 |
| 布局 | 居中大头像 + 三行输入 | 无大头像；号码组合框+申请号码/密码+忘了密码 两行 |
| 验证码 | 有（4 位字母） | **该年代登录无验证码**（验证码属网页/异地场景） |
| 选项 | 记住密码/自动登录/隐身登录 | 自动登录/隐身登录（两项）+ 高级设置 |
| 登录中 | 半透明遮罩 + 弹跳企鹅 | 独立窄条窗 + 进度条 + 取消登录 |
| 结果 | 必然"版本过旧"死胡同 | 应进入主面板（#119 已要求把彩蛋改为可配置） |
| 字体/按钮 | Tahoma、自定义渐变按钮 | 宋体 12px、`#FFF→#9FD4FF/#00558E` 系按钮 |

## 5. 声音与提示行为 [确凿]

原版 QQ 安装目录 `sound\` 的经典音效及对应事件：

| 文件 | 声音 | 事件 | 本仓库现状 |
|---|---|---|---|
| `msg.wav` | "滴滴滴滴滴滴"（摩托罗拉寻呼机录制） | 好友消息 | `soundManager.qqMessage()` 已有合成近似，未接线 |
| `Global.wav` | 敲门"咚咚" | **好友上线** | `qqOnline()` 已有合成近似，未接线；注意现有 `qqKnock()` 被登录用作"敲门"，语义上敲门=上线而非登录 |
| `system.wav` | "咳、咳"两声咳嗽（腾讯后来注册了声音商标） | 系统消息/加好友请求 | 无 |
| `folder.wav` | — | 群消息 | 无（MVP 可不做群聊） |

提示行为链（收到消息，窗口未打开/未聚焦时）：
1. **托盘企鹅图标被替换为发送者头像，与空白交替闪烁**（约 500ms 周期）；
   单击托盘打开最早未读的聊天窗并停闪；多人未读则继续闪下一位头像。
   右键菜单有"关闭头像闪动"。
2. 好友列表内该好友头像同步闪烁。
3. 已打开未聚焦 → XP 任务栏按钮橙色闪烁（系统行为，引擎 `flashWindow` 已有）。
4. **好友上线**：敲门声 + 从托盘处滑出/淡入小面板（约 180×70px）
   "您的好友 昵称(QQ号) 上线了。"，数秒淡出，点击直接开聊天窗
   （引擎 `notify({ anchorId })` 气球已支持锚定托盘图标）。
5. **窗口抖动**：QQ2007II 才有——按 §0 建议做成配置项彩蛋。触发后整窗小幅
   快速位移约 1s，聊天区插入系统行"您/对方发送了一个窗口抖动"。

## 7. 素材与版权策略

- 表情/系统头像/音效/皮肤图版权归腾讯；参照的复刻项目自身也声明"仅供学习交流"。
- 本项目现行做法（QQ 登录、360、迅雷等 zh 文化包）是**自绘致敬**而非搬运素材，
  建议延续：**渐变/按钮/布局全部用上表 CSS 值实现（零图片）**；头像用自绘的
  40×40 卡通集（8–12 个即可覆盖剧本需要）；表情用自绘 16px 黄脸 SVG；音效沿用
  soundManager 的合成近似（已有 `qqMessage`/`qqOnline`，补一个咳嗽声 `qqSystem`）。
- 唯一建议引入的"数据"是**表情码对照表**（`/wx`→微笑 等映射本身是事实，无版权）。

## 8. 与 #119 实现的映射（引擎侧落点）

| #119 要求 | 考据结论 → 落点 |
|---|---|
| 好友列表窗 | §1 规格；`QQMessenger` 注册单例窗 190×580 不可调；分组+计数+灰度离线+签名 |
| 聊天窗 | §2 规格；两行式消息流（蓝/绿昵称行）是核心保真点，不做气泡 |
| 脚本化内容 | buddies/scripts 进 zh 文化包或场景 JSON；per-buddy `reply: { script } \| { provider: 'chat' }`（#148 接口形状，评论已定） |
| 来消息流 | 托盘头像闪烁 = `TrayContext.update()` 轮换 icon 已可实现；任务栏闪 = `flashWindow` 已有；声音 = `qqMessage()` 接线 |
| 上线提醒 | `qqOnline()`（敲门）+ `notify({ anchorId: 'qq' })` 气球，文案"您的好友 xxx 上线了" |
| 打字效果 | 流式 `AsyncIterable` → 逐字渲染（#148 的 streaming→typing 约定）；脚本消息用 `typingMs` 停顿字段 |
| emojiRenderer | §3：升级为 `/缩写码 + [中文名]` 双解析、输出 16px 经典黄脸；两个 QQ 音效与它一起转正 |
| 事件/句柄 | 新增 `qq:open / qq:message / qq:reply`（扩展 `src/events.ts` union，#130 语法）+ `XPHandle` 上 `qq.sendMessage(buddyId, text, opts)`（或经 #84 场景动作 `qqMessage`） |
| 登录接线 | §4：重做 `QQLogin` 为真实 QQ2006 结构（去验证码、加登录中窄条窗），成功 → 打开主面板；"版本过旧"彩蛋降级为配置项 |
| 年代守则 | 抖动默认关（2007II 特性）；界面绝不出现 2008+ 元素（气泡、状态"Q我吧"文案等按目标年代取舍） |
