# 《山月无声》故事大纲

## 核心概念

**类型**: 社会派悬疑互动解谜游戏

**主题**: 当制度的缝隙吞噬个体，真相便沉没于系统的沉默与个体的怯懦之中

**叙事结构**: 双时间线（2015-2016过去 + 2026现在）+ 玩家视角层层递进

**玩家视角**: 2026年的夏灯，通过父亲遗留的XP电脑调查十年前的悲剧真相

**社会派推理特色**: 从个体悬疑（谁是凶手）到系统揭露（高考移民产业链）

**游戏时长**: 3.5-4小时

---

## 核心人物

- **林晓宇** (2000-2016) - 摄影少年，真相记录者，象征"无声"
- **陈默** (1999-) - 技术少年，被迫参与替考，九年自我囚禁后自首，象征"山"
- **夏灯** (1999-) - 教导主任之女，玩家角色，真相追寻者，象征"灯"
- **夏建国** (1965-2025) - 县一中教导主任，知情但沉默的记录者、帮凶
- **张雨** (1998-) - 林晓宇暗恋的学姐，高考移民受害者，调查的起点

*详细人物设定见 [人物.md](人物.md)*

## 故事梗概

### 第一幕：友谊与裂变（2015年秋-冬）

**山顶约定（2015年9月）**

- 三人在山顶成立"山顶事务所"，讨论社会派推理小说，探讨记录、守护与罪的边界
- 陈默奶奶住院，需要手术费3万元

**真相的种子（2015年8月-11月）**

- **8月，张雨的遭遇**：林晓宇暗恋的学姐张雨高考589分，超国家专项线4分，却因名额被占只能去二本
- 林晓宇在照相馆看到张雨和母亲哭泣，看到"国家专项不予录取"的通知
- **调查的起点**：林晓宇开始调查为什么国家专项名额被占用
- 11月，林晓宇在照相馆发现可疑的"证件照客户"，开始跟踪拍摄
- 陈默因奶奶病重被王虎招募（给5万元），利用机房权限篡改教务系统学籍数据
- 夏灯察觉父亲异常，但选择服从"不要惹事"的告诫

**友谊破裂（2015年12月）**

- 林晓宇拍到关键证据，在群里分享疑点
- 陈默用技术警告林晓宇清理EXIF信息
- 校园贴吧开始出现针对林晓宇的网络暴力（"偷拍狂"谣言）
- 夏灯在父亲要求下疏远林晓宇

### 第二幕：坠落与掩盖（2016年初春）

**对峙（2016年2月14日）**

- 林晓宇发现陈默参与，图书馆当面质问
- 陈默痛苦坦白："系统想改一个名字、抹掉一个人，就像删除一个文件"
- 林晓宇理解陈默处境，但决定继续对抗
- 林晓宇写下加密日志，上传证据照片

**悲剧（2016年2月15日）**

- 陈默想陪林晓宇去见王虎，提前到达烂尾医院
- 校霸赵威受王虎指使，带人抢夺林晓宇书包
- 追逐至烂尾医院楼顶，争夺中林晓宇失足坠楼身亡
- 陈默试图阻止，但无力回天，眼镜在拉扯中掉落

**集体沉默**

- 王虎团伙散布谣言，警方快速定性"意外坠亡"
- 葬礼冷清，夏建国未到场，夏灯被禁止参加
- 陈默躲在远处目睹，然后呕吐逃离
- 县城在高考压力下迅速遗忘
- 真相碎片散落在互联网的角落

### 第三幕：蛰伏与变迁（2016-2025）

**各自的流放**

- **陈默**: 切断联系，外地求学，成为深圳程序员。2020-2024年因奶奶病情恶化需透析，被迫放弃自首，继续隐忍，保存所有证据
- **夏灯**: 考入师范大学教育学专业，潜意识探究"为什么系统会吞噬个体"
- **夏建国**: 调离管理岗位，继续在加密日志里记录，2025年底确诊肝癌晚期

### 第四幕：打捞与救赎（2026年）

**遗言的触发（2026年1月初）**

- 夏建国临终遗言："电脑…D盘…密码是你生日…我对不起…那孩子…他拍到了…"
- 夏灯回到老家，打开父亲遗留的XP电脑
- 发现加密工作日志和九年来的记录

**追寻真相（2026年1-2月）**

**阶段一：个体悬疑**（怀旧、困惑）

- 登录QQ，浏览"山顶事务所"群聊记录
- 访问林晓宇QQ空间，看到从阳光到沉默的转变
- 玩家疑问：林晓宇发现了什么？怎么死的？

**阶段二：证据指向陈默**（愤怒、背叛）

- 破解林晓宇QQ空间加密相册（密码：曝光三角28125400）
- 查看照片：现金交易、机房窗户、图书馆对峙
- 解密父亲文件夹，发现：
  - 陈默11.23导出学生名单、12.02凌晨在机房
  - 警方记录：现场有黑框眼镜（陈默戴眼镜）
  - 短信截图：陈默告诉赵威林晓宇的行踪
  - 林晓宇最后私聊：担心"陈默会告诉他们"
- 与陈默邮件往来，陈默承认在现场、眼镜是他的
- 玩家推论：**陈默至少是共犯，甚至可能是凶手**

**阶段三：真相反转**（震撼、理解）

- 破解林晓宇加密日志（密码：camera3rdeye）
- 现代诗隐藏信息："我会被他们杀死王虎在操控替考"
- 林晓宇日志：陈默是被迫的（奶奶手术费），林晓宇理解他
- 短信完整版：陈默警告赵威别去，想保护林晓宇
- 陈默完整证词：去现场是为了救林晓宇，但失败了
- 玩家认知推翻：**陈默不是凶手，是另一个受害者**

**阶段四：系统揭露**（沉重、反思）

- 解密父亲文件夹第二层（密码：20151123）
- 完整的系统操作记录：
  - 2013-2015三年，共18人（每年6人），总收益约2000万
  - 保护伞网络：教育局+公安局+学校
  - 受害者名单：张雨、周磊、方欣等人生轨迹被改变
- 林晓宇日志第二部分：
  - 张雨的故事（589分→二本，妈妈是清洁工）
  - "我们在系统的缝隙里挣扎，然后被吞噬"
- 陈默决定自首，说出全部真相

**真相被看见**

- 三天后，陈默在深圳市公安局自首
- 一周后，省级媒体刊发调查报道《被遗忘的少年：一桩十年悬案的数字重建》
- 县公安局重启调查，完整真相被还原
- 林晓宇的母亲："儿子，你没有做错，是妈妈来晚了"

**尾声（2026年清明）**

- 陈默因主动投案、配合调查，被判有期徒刑一年，缓刑一年
- 夏灯和陈默站在林晓宇墓前
- 陈默："晓宇，对不起，我来晚了"
- 夏灯："晓宇，我们终于让你被看见了"

**山月无声，但光终究会找到裂缝**

## 核心冲突

1. **个体 vs 系统**: 普通少年的偶然发现 vs 利益链条的系统性掩盖
2. **记录 vs 遗忘**: 林晓宇的镜头和文字 vs 县城的集体失忆
3. **生存 vs 良知**: 陈默的家庭压力 vs 对朋友的愧疚
4. **沉默 vs 发声**: 夏灯和夏建国的"知情不言" vs 最终的追查与公开
5. **个体悲剧 vs 系统性腐败**: 从"谁害死了林晓宇"到"高考移民产业链如何吞噬普通人"

## 社会派推理的层次结构

```
第一层：个体悬疑
问题：林晓宇是怎么死的？
玩家关注：死因真相、陈默可疑

第二层：证据指向
问题：陈默是凶手吗？
玩家关注：陈默的行为、现场证据、短信截图

第三层：真相反转
问题：为什么会这样？
玩家关注：陈默动机、林晓宇理解、完整真相

第四层：系统揭露
问题：这背后是什么？
玩家关注：高考移民产业链、18人、2000万、受害者、保护伞
```

**关键设计**：玩家不是一开始就知道"这是关于高考移民的故事"，而是先关注"谁害死了林晓宇"，最后才发现"原来林晓宇是被整个系统吞噬的"。

## 叙事特色

- **文学指涉系统**: 《白夜行》《桶川跟踪狂杀人事件》《为了N》等作品贯穿始终
- **双时间线交织**: 2026年的调查逐步揭开2015-2016年的真相
- **数字考古叙事**: 通过XP电脑访问QQ空间、贴吧、网页快照等互联网遗迹
- **陈默的心理弧线**: 从逃避到面对，从被动到主动，展现救赎的艰难过程
- **无英雄叙事**: 没有完美的正义使者，所有人都在现实压力下妥协，救赎来自迟到的勇气
- **社会派深度**: 从个体悬疑逐步升华到对教育不公、阶层固化、系统性腐败的揭露

## 红鲱鱼系统

### 红鲱鱼#3：陈默=凶手？（核心误导）

**误导证据链**：

1. 父亲日志：陈默技术操作（11.23导出、12.02凌晨导入）
2. 相册照片：12.02凌晨机房有人
3. 父亲日志：2.14林晓宇和陈默吵架，第二天林晓宇就死了
4. 警方记录：现场有黑框眼镜（陈默戴眼镜）
5. 警方记录：证人看到"戴眼镜的男生"
6. 短信截图（断章取义）：陈默告诉赵威行踪，"你知道该怎么做"
7. 林晓宇最后私聊：担心"陈默会告诉他们"
8. 陈默邮件：承认在现场、眼镜是他的、短信是他发的

**推翻证据**：

1. 林晓宇日志：陈默是被迫的，林晓宇理解他
2. 短信完整版：陈默警告赵威别去，想保护林晓宇
3. 陈默完整证词：去现场是为了救林晓宇

**情感冲击**：从愤怒、背叛到理解、同情

### 红鲱鱼#1：情绪自杀（简单铺垫）

**误导证据**：

- QQ空间说说越来越阴郁，引用《白夜行》等抑郁文学
- 贴吧"偷拍狂"网络暴力
- 警方结论"心理压力过大，意外坠楼"

**作用**：

- 让玩家前20分钟以为是"网络暴力导致的自杀"
- 快速铺垫，不是重点

### 红鲱鱼#2：林晓宇的秘密恋情（轻度误导+调查起点）

**线索**：

- 群聊和空间说说中反复提到"在图书馆看到她"
- "589分，她一定会去很好的大学"
- "今天看到她哭了，为什么589分只能去二本？"

**双重作用**：

- 增加人物立体感（青春暗恋）
- 实际上是张雨学姐，也是林晓宇调查的起点
- 为"偷拍狂"谣言提供基础（他确实拍过张雨）

## 情感内核

这不是一个关于天才少年对抗黑暗的故事，而是关于**普通人如何在记忆的负重下寻求赎罪与和解**的故事。它探讨：

- 当系统失灵时，个体的记录是否有意义？
- 为了生存而妥协的罪，能否被原谅？
- 知情但沉默的旁观者，如何面对良心的审判？
- 迟到九年的真相，是否还有价值？
- **一个普通少年的死亡，如何揭示整个教育系统的腐败？**
- **高考移民产业链如何改变无数家庭的命运？**

**答案是：有。因为被看见本身，就是对沉默最有力的反抗。**

## 社会议题呈现

### 高考移民产业链

**真实机制：利用国家专项计划**

本县为国家级贫困县，享有"国家贫困地区定向招生专项计划"：

- 重点大学为本县划拨专项名额（如10个）
- 要求：具有本县连续3年学籍+户籍
- 分数线比普通批次低20-30分（如：985普通线615分，本县专项线585分）

**操作流程**：

1. 外省/外市中介招揽高分考生（针对富商/官员子女，收费80-120万/人）
2. 王虎在本地运作（打通教育局、公安局、学校）
3. 陈默篡改教务系统，伪造学籍（从高一开始伪造3年记录）
4. 学校开具虚假就读证明（夏建国等人签字）
5. 教育局副局长快速审批
6. 外省考生以"本县考生"身份参加高考，利用国家专项降分政策录取

**规模**：2013-2015年，3年共操作18人（每年6人），总收益约1800-2000万

**保护伞系统**：

- 教育局/招办：40%（约800万）- 关键审批
- 公安局/户籍：30%（约600万）- 伪造户籍、平事
- 王虎/中介：20%（约400万）- 组织、脏活
- 学校领导/具体执行：10%（约200万）- 配合造假

### 被挤占的未来

**核心矛盾**：国家专项计划本是帮助贫困县考生的扶贫政策，却被权贵利用，真正的贫困生反而被挤出名额。

**受害者案例**：

- **张雨**（林晓宇暗恋的学姐）：
  - 2015年高考589分，超国家专项线4分（专项线585分）
  - 本应通过国家专项进入985大学
  - 因6个假学籍占用名额，专项落榜，只能录取二本
  - 母亲是县医院清洁工，月薪1500元，单亲家庭
  - 复读费用是妈妈大半年工资，最终放弃复读
  - 现状：某县城文员，月薪3000元

- **周磊**：
  - 2015年高考582分，超国家专项线（本应录取）
  - 因名额被占，录取二本师范
  - 现状：某乡村小学教师，月薪4500元
  - "我不后悔当老师，但我后悔失去了选择的权利"

- **方欣**：
  - 2015年高考578分，本应通过专项进入一本
  - 因名额被占，录取三本民办
  - 父母农民，负债10万供她上学（民办学费2万/年，公立一本只需6000/年）
  - 现状：打工还债

**主题升华**：

- 20万，可以让张雨复读13年
- 20万，可以救陈默奶奶6次
- 但这些钱，流进了权力掮客的口袋

**扶贫政策被权贵利用，真正的穷人被系统吞噬。**

## 玩家体验设计核心

**情感曲线**：

```
怀旧 → 困惑 → 愤怒 → 震撼 → 理解 → 沉重 → 希望
```

**认知变化**：

```
意外坠亡 → 林发现了什么？ → 陈默是凶手？ → 陈默也是受害者 → 系统性腐败
```

**悬疑递进**：

```
个体悬疑 → 证据指向 → 真相反转 → 系统揭露
```

这才是真正的社会派推理——从个体悲剧到系统性问题的升华。
