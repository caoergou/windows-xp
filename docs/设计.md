# 《山月无声》游戏设计

*素材规格与叙事控制*

---

## 设计概览

### 游戏形式

模拟Windows XP桌面环境，玩家通过以下方式调查真相：

- 本地文件系统（文件管理器）
- QQ客户端（群聊、好友、空间）
- 模拟浏览器（贴吧存档、网页快照）
- 邮件客户端（与陈默往来）

### 时间跨度

- **过去线**：2015-2016年（通过数字遗迹呈现）
- **现在线**：2026年1-2月（玩家操作时间）

### 游戏时长

3小时（180分钟）

- 解密：75分钟
- 阅读：105分钟

---

## 素材组织结构

### 素材分类

```
assets/
├── chat-logs/          # 群聊记录（JSON）
├── qq-space/           # QQ空间内容（JSON）
├── emails/             # 陈默邮件（JSON）
├── documents/          # 本地文档（TXT + JSON元数据）
├── photos/             # 照片（JPG + JSON EXIF）
├── web-archives/       # 网页存档（JSON）
└── 清单.md             # 素材制作清单
```

### 数据格式规范

所有结构化数据使用JSON格式：

- 群聊记录：JSON数组
- QQ空间：JSON对象
- 邮件：JSON数组
- 照片EXIF：JSON元数据文件
- 网页存档：JSON结构

纯文本内容（父亲日志、林晓宇诗歌）：TXT格式

### 3. 邮件

**文件**：`assets/emails/chenmo-emails.json`

**数据结构**：

```json
{
  "correspondence": [
    {
      "id": "email_001",
      "from": "chenmo.dev@gmail.com",
      "to": "xiadengnj@163.com",
      "date": "2026.01.05",
      "subject": "惊闻噩耗",
      "body": "...",
      "trigger": "game_start",
      "status": "unread"
    }
  ]
}
```

**触发机制**：

| 邮件 | 触发条件 | 情感状态 |
|------|---------|---------|
| 邮件1 | 游戏开始 | 慰问、试探 |
| 邮件2 | 玩家回复邮件1 | 回避、引导 |
| 邮件3 | 破解相册后 | 痛苦、坦白部分 |
| 邮件4 | 看完父亲日志后 | 承认在现场 |
| 邮件5 | 看完短信完整版后 | 完整证词 |
| 邮件6 | 读完所有证据 | 决心、告别 |

**自动交互机制**：

- **自动草稿**：当玩家在游戏中解锁关键证据（如破解相册、看完日记）后，打开邮件客户端，系统会自动根据当前进度生成一封发给陈默的草稿。
- **一键发送**：玩家点击“发送”即可。
- **触发回复**：发送后，经过短暂的模拟延迟（或立即），收到陈默的回复，推动剧情进入下一阶段。

**设计原则**：

- 邮件只提供方向性引导，不直接给密码
- 情感递进：试探→回避→痛苦→坦白→告别
- 不发送证据包附件

### 7. 调查报道

**文件**：`assets/documents/investigation-report.txt`

**内容结构**：

```
第一部分：一个少年的死亡

```

**关键内容设计**：

**记者手记：扶贫政策的异化**（约1500字）

> **国家专项计划的初衷**
>
> 2012年，国家启动"贫困地区定向招生专项计划"（简称国家专项），旨在为贫困县考生提供进入重点大学的机会。政策规定：重点大学为国家级贫困县划拨专项名额，录取分数线比普通批次低20-30分。
>
> 这本是一项教育公平政策，让寒门子弟有机会通过高考改变命运。
>
> **政策的异化**
>
> 但在这个县城，2015年的6个国家专项名额中，有5个被外省富商子女占用。唯一的本地考生是张雨——她妈妈是县医院清洁工，月薪1500元，单亲家庭。
>
> 张雨考了589分，超国家专项线4分（专项线585分），本应通过国家专项进入985大学。但因为5个假学籍占用名额，她的专项资格被取消，只能录取二本。
>
> **数字的讽刺**
>
> - 一个外省考生花120万买走一个国家专项名额
> - 120万，可以让张雨复读13年（每年复读费9000元）
> - 120万，可以救陈默奶奶40次（每次手术费3万元）
> - 120万，是张雨妈妈66年的工资（月薪1500元）
>
> 但这些钱，流进了权力掮客的口袋。
>
> **更深的讽刺**
>
> 那个花120万的家庭，孩子只考了560分，比张雨低29分。
>
> 如果没有国家专项，这个孩子只能上二本。但通过伪造学籍，他以"本县考生"身份参加高考，利用国家专项降分政策，进入了985大学。
>
> 而真正的本县考生张雨，589分，只能去二本。
>
> **扶贫政策被权贵利用，真正的穷人被系统吞噬。**
>
> 这不是个案。根据调查，2013-2015年三年间，该县共有18人通过类似手段占用国家专项名额，涉案金额约2000万元。
>
> 18个外省富家子弟，挤掉了18个本地寒门学子的未来。

**设计用途**：

- 游戏结局呈现
- 完整展示社会议题
- 受害者采访（张雨、周磊、方欣）
- 主题升华
- **记者手记**：揭示国家专项计划被权贵劫持的制度讽刺

- Windows XP 记事本样式
- 自动打开并置顶
- 打字机效果：逐字显示，带有打字音效（可选）
- 打字完成后，文件自动保存到桌面
- 玩家可以随时打开桌面上的笔记文件回顾

**技术实现要点**：

```javascript
// 伪代码示例
function openInvestigationNote(filename, content, typingSpeed = 50) {
  // 1. 打开记事本窗口
  const notepad = openNotepad(filename);

  // 2. 逐字输入内容
  typeText(notepad, content, typingSpeed);

  // 3. 输入完成后自动保存到桌面
  saveToDesktop(filename, content);

  // 4. 更新游戏进度标记
  markProgressFlag(`investigation_note_${filename}`);
}
```

**叙事作用**：

- **主动推理**：不是被动接受陈默的邮件，而是夏灯主动分析证据
- **红鲱鱼强化**：笔记02明确表达对陈默的怀疑，增强误导效果
- **情感递进**：从怀疑→愤怒→理解→决心
- **进度可视化**：玩家可以通过桌面上的4个笔记文件看到调查进展
- **沉浸感**：打字机效果让玩家感觉"夏灯正在思考"

**与邮件系统的配合**：

- 调查笔记：夏灯的主动推理（内部视角）
- 陈默邮件：外部信息补充（外部视角）
- 两者结合，形成完整的叙事推动机制
