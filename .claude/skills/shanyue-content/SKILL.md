# 山月无声·内容创作

为叙事游戏《山月无声》创作符合角色真实性的 QQ 空间内容（说说、日志、相册）。

## 快速开始

当用户请求创作内容时：

1. **确认需求**：角色、内容类型、时间段、数量
2. **加载角色指南**：阅读 `references/characters/{角色名}.md`
3. **查阅数据规范**：参考 `references/data_specs.md` 了解 JSON 格式
4. **生成内容**：创作真实、符合时间线的内容
5. **验证**：检查 JSON 格式、情感弧线、故事一致性

## 内容类型

### 1. QQ 空间说说

类似微博/推特的短动态。用于：

- 日常生活分享
- 情感表达
- 书籍/电影引用
- 照片分享与配文

**创作时长**：每次 10-15 条说说
**关键考量**：情感随时间的递进变化

### 2. QQ 空间日志

长篇博客文章。用于：

- 摄影随笔
- 个人反思
- 加密告白（含密码提示）
- 调查记录

**创作时长**：每次 2-5 篇日志
**关键考量**：文学性与角色声音

### 3. 相册

带描述的照片集合。用于：

- 日常摄影作品集
- 加密证据相册
- AI 生图提示词

**创作时长**：每次 1-2 个相册，每个 5-12 张照片
**关键考量**：视觉叙事与 AI 提示词质量

## 工作流程

### 步骤 1：角色语境

询问用户明确：

- **角色**：林晓宇、陈默或夏灯
- **时间段**：处于哪个情感阶段？
  - 林晓宇：阳光期 (2014.09-2015.08)、困惑期 (2015.09-11)、沉默期 (2015.12-2016.02)
  - 陈默：参与前、共犯期、自我囚禁 (2016-2025)、救赎期 (2026)
  - 夏灯：顺从期 (2015)、觉醒期 (2026)
- **内容类型**：说说、日志还是相册
- **数量**：需要多少条

### 步骤 2：加载参考资料

**角色真实性参考**，阅读：

- `references/characters/lin_xiaoyu.md` - 摄影学生，理想主义的真相追寻者
- `references/characters/chen_muo.md` - 技术爱好者，道德挣扎者
- `references/characters/xia_deng.md` - 主任之女，从被动旁观到主动调查

**数据格式参考**，查阅：

- `references/data_specs.md` - 完整 JSON 模式要求

### 步骤 3：生成内容

遵循角色专属指南：

**林晓宇的声音：**

- 短句，诗意措辞
- 引用推理小说（《白夜行》《为了N》）
- 摄影术语（光圈、快门、ISO）
- 情感弧线：乐观 → 焦虑 → 绝望

**陈默的声音：**

- 技术术语（数据库、系统、权限）
- 直接、少废话
- 2026 年邮件：克制、愧疚、试探

**夏灯的声音：**

- 2015年：礼貌、克制、顺从
- 2026年：分析性、共情、调查性

### 步骤 4：应用数据规范

**关键要求**（详见 `references/data_specs.md`）：

```json
{
  "time": "YYYY-MM-DD HH:mm:ss",  // 不是 YYYY/MM/DD
  "photos": ["/photos/album/file.jpg"],  // 以 / 开头
  "locked": true,  // 加密内容
  "password": "28125400",  // 必须与提示匹配
  "hint": "提示:曝光三角(光圈-快门-ISO)"
}
```

**密码设计模式：**

- 曝光三角：`f/2.8 + 1/250 + ISO400 → "28125400"`
- 英文短语：`"camera 3rd eye" → "camera3rdeye"`
- 日期：`1999-03-15 → "19990315"`

### 步骤 5：质量检查

验证：

- [ ] JSON 语法正确
- [ ] 时间格式：`YYYY-MM-DD HH:mm:ss`
- [ ] 路径以 `/` 开头
- [ ] 角色声音一致
- [ ] 情感弧线符合时间线
- [ ] 加密内容包含 `locked`、`password`、`hint`
- [ ] 无提前剧透（玩家不应在前期知道"高考移民案"）

## 信息隐藏原则

**阶段一内容**（早期调查）：

- 展示：模糊的"交易"、陈默可疑行为
- 隐藏：高考移民、15名受害者、产业链

**阶段二内容**（证据指向）：

- 展示：陈默技术操作、现场证据
- 隐藏：陈默动机、完整短信、受害者故事

**阶段三内容**（真相反转）：

- 展示：陈默动机、完整短信、林晓宇的理解
- 隐藏：完整产业链规模、保护伞网络

**阶段四内容**（系统揭露）：

- 全部揭示
