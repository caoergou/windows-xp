# 《山月无声：云山一中调查录 2015》品牌设定

*游戏命名、视觉识别与品牌表达*

---

## 命名体系

### 正式名称

**山月无声：云山一中调查录 2015**

### 命名逻辑

| 组成部分 | 作用 | 说明 |
|---------|------|------|
| 山月无声 | 主题锚定 | 核心文学意象，体现故事的诗意深度和社会派气质，呼应结尾「山月无声，但光终究会找到裂缝」的主题升华 |
| 云山一中 | 地点锚定 | 虚构的县城高中，「云山」暗示偏远山区县城，「一中」是中国县城最典型的学校命名 |
| 调查录 | 类型标识 | 一看即知是调查/推理类游戏，「录」字有档案、记录的含义，呼应游戏中翻查文件、聊天记录的核心玩法 |
| 2015 | 时间锚点 | 悲剧发生的年份，制造「那一年发生了什么」的悬念，也暗示这是一桩旧案 |

空格分隔形成层次分明的视觉感受：先有诗意主题，再有具体内容，符合游戏从个体悬疑到系统揭露的叙事节奏。

### 名称变体

| 场景 | 使用形式 |
|------|---------|
| 正式全称 | 山月无声：云山一中调查录 2015 |
| 口语简称 | 山月调查录 / 调查录2015 |
| 英文全称 | The Silent Mountain Moon: Yunshan High Case Files 2015 |
| 英文简称 | Silent Moon: Case Files 2015 |
| 代码/包名 | shanyue-yunshan-2015 |
| URL slug | shanyue-yunshan-2015 |

### 「山月无声」的核心地位

「山月无声」作为游戏的核心主题和文学意象，在品牌层面具有重要地位：

- **标题前置**：直接体现在游戏名称中，强化主题识别
- **通关结语**：大纲结尾「山月无声，但光终究会找到裂缝」作为游戏通关后的收束文字
- **游戏内引用**：出现在林晓宇的QQ空间说说、日志、或山顶事务所的群聊讨论中，作为三人对社会派推理的感悟
- **品牌视觉**：可作为辅助图形元素，增强品牌的文学性

### 为什么这样改？

原标题「云山一中调查录：2015」功能完整但缺乏情感温度。新标题：

1. 保留了原书名的诗意内核，与故事主题形成呼应
2. 信息层次更清晰：主题 → 地点/类型 → 时间
3. 强化了品牌的文学性，在同类游戏中更具辨识度
4. 符合玩家对独立游戏的期待，兼具严肃性和艺术性

## 核心意象

### 三层象征体系

游戏的视觉和叙事围绕三个核心意象展开：

| 意象 | 象征 | 对应人物 | 视觉表达 |
|------|------|---------|---------|
| 灯 | 真相、希望、照亮黑暗 | 夏灯（玩家角色） | 琥珀色/暖光色，Landing Page 中「2015」的颜色 |
| 档案/记录 | 证据、记忆、不可磨灭 | 林晓宇（记录者） | 等宽字体、打字机效果、纸张质感 |
| 沉默/黑暗 | 系统性掩盖、集体失语 | 陈默 / 夏建国 / 整个县城 | 深蓝色调、CRT扫描线、低对比度文字 |

### 核心视觉隐喻

**旧电脑 = 时间胶囊**

整个游戏发生在一台2015年的Windows XP电脑上。这台电脑本身就是品牌最核心的视觉符号：

- 它是父亲的遗物（情感连接）
- 它保存了十年前的真相（叙事载体）
- 它的界面就是游戏界面（形式即内容）
- 「开机」这个动作 = 「重启调查」（玩法隐喻）

## 视觉识别

### 色彩体系

#### 主色板

| 名称 | 色值 | 用途 |
|------|------|------|
| 深夜蓝 | `#1a1a2e` | Landing Page 背景主色，代表沉默与未知 |
| 暗蓝 | `#16213e` | 背景渐变中间色 |
| 深海蓝 | `#0f3460` | 背景渐变终止色 |

#### 强调色

| 名称 | 色值 | 用途 |
|------|------|------|
| 琥珀色 | `#d4a853` | 「灯火」象征色，用于「2015」数字、关键UI高亮、hover状态 |
| 暗金 | `#b8942e` | 琥珀色的暗调变体，用于次要强调 |

#### 文字色

| 名称 | 色值 | 用途 |
|------|------|------|
| 亮白 | `#f0f0f0` | 主标题 |
| 冷灰 | `#b0bec5` | 叙事正文 |
| 暗灰 | `#8899aa` | 辅助文字、提示 |
| 深灰 | `#6a7a8a` | 氛围文字、声明 |

#### 色彩使用原则

1. **冷暖对比**：整体冷色调（深蓝/灰）中，琥珀色作为唯一暖色出现，代表「灯」的意象。琥珀色的使用要克制，只在关键位置出现（「2015」、重要按钮hover、游戏内关键发现时刻）
2. **明暗层次**：信息越重要，文字越亮；氛围性文字用低对比度，不抢注意力
3. **游戏内色彩**：进入XP桌面后，色彩体系切换为Windows XP原生配色（蓝色任务栏、绿色开始按钮、白色窗口），Landing Page 的深蓝色调不延续到游戏内

### 字体体系

| 用途 | 字体 | 说明 |
|------|------|------|
| 中文标题「山月无声」 | 楷体/行书 | 体现诗意和文学性 |
| 中文副标题「云山一中调查录」 | SimSun / STSong（宋体） | 宋体有档案感、正式感，呼应「调查录」的文书属性 |
| 数字「2015」 | Courier New / monospace | 等宽字体的打字机感，呼应旧电脑、文件记录、档案编号 |
| 英文副标题 | Tahoma | XP系统默认字体，保持与游戏内的一致性 |
| 叙事正文 | Tahoma / Microsoft YaHei | 可读性优先 |
| 游戏内界面 | Tahoma / SimSun | 严格遵循Windows XP原生字体 |

### Logo

纯文字Logo，层次分明：

```
山月无声
    云山一中调查录 2015
```

- 第一行「山月无声」：楷体/行书，白色（`#f0f0f0`），字间距适中，体现诗意
- 第二行「云山一中调查录 2015」：「云山一中调查录」用宋体，「2015」用Courier New等宽字体，琥珀色（`#d4a853`），形成视觉对比
- 两行之间无装饰线，靠字体和颜色的差异形成层次

不使用图形Logo的原因：

- 游戏的核心视觉就是XP桌面本身，不需要额外的图形符号
- 文字Logo更贴合「档案/调查录」的调性
- 简化跨平台适配（网页favicon可直接用「云」或「2015」的文字截取）

## Landing Page 表达

### 标题区域

```
山月无声
    云山一中调查录 2015

THE SILENT MOUNTAIN MOON: YUNSHAN HIGH CASE FILES
```

- 主标题「山月无声」：楷体/行书，48px，白色，letter-spacing: 6px
- 副标题「云山一中调查录」：宋体，36px，亮白色，letter-spacing: 8px
- 「2015」：Courier New，40px，琥珀色（`#d4a853`），letter-spacing: 16px
- 英文副标题：Tahoma，13px，`#8899aa`，letter-spacing: 4px

### 叙事文案

```
2026年。父亲去世后，你在旧物中发现了一台尘封的电脑。
十年前的文件、聊天记录、被加密的日志......
那些被刻意遗忘的，正在等你打开。
```

保持现有文案不变。这段文字精准传达了游戏的设定（2026年、父亲遗物、旧电脑）、玩法（翻查文件、聊天记录、解密）和情感基调（遗忘与重新发现）。

### 游戏提示

混合风格 -- 前两条实用建议，后两条叙事氛围：

```
推荐使用 Chrome / Edge 浏览器，全屏体验更佳
准备好纸笔 -- 记录你发现的每一条线索
留意时间、细节与矛盾之处
不要轻信任何单一来源的信息
```

- 前两条：`#8899aa`，正常字体，提供实用信息
- 后两条：`#6a7a8a`，斜体，从实用过渡到氛围，最后一条既是游戏提示也是故事主题的暗示

### 虚构声明

```
本作品纯属虚构，所有人物、事件、地点均为艺术创作。如有雷同，纯属巧合。
```

- 位置：页面底部，绝对定位
- 字号：10px，颜色 `#5a6a7a`
- 只声明虚构，不提及具体敏感题材

### 底部信息

```
一款社会派悬疑解谜游戏 / 预计游戏时长 3-4 小时
```

打字机效果逐字显示，延迟2.5秒开始。

### 按钮

- 首次访问：「开启电脑」
- 回访玩家：「继续调查」+ 闪烁提示「检测到未完成的调查记录」

## 标语体系

### 主标语

> 十年前的真相，藏在这台旧电脑里。

用于 Landing Page 底部打字机文字之外的宣传场景（社交媒体分享、应用商店描述等）。

### 场景标语

| 场景 | 标语 |
|------|------|
| 社交媒体分享 | 2015年，云山县发生了一件事。所有人选择了沉默。 |
| 游戏简介 | 一台父亲遗留的旧电脑，十年前被加密的文件，一个被定性为「意外」的坠楼事件。 |
| 玩法描述 | 每一条聊天记录，都是一块拼图。 |
| 情感向 | 有些文件，删不干净。 |
| 主题向 | 被看见，就是对沉默最有力的反抗。 |

### 标语使用原则

1. 不在标语中出现「高考移民」「产业链」「腐败」等剧透性关键词
2. 标语的信息层级与游戏阶段一对应 -- 对外宣传只透露阶段一的信息（个体悬疑：一个少年的死亡）
3. 「社会派悬疑」作为类型标签使用，不在标语正文中出现

## 域名

### 推荐方案

| 优先级 | 域名 | 说明 |
|--------|------|------|
| 首选 | `shanyue2015.cc` | 10字符，包含「山月」拼音和「2015」，直接对应游戏名核心信息，.cc 约 $3-5/年 |
| 备选1 | `shanyue2015.ink` | 同上，.ink 呼应游戏中文字记录/证据的主题 |
| 备选2 | `silentmoon2015.cc` | 英文向，国际化友好，15字符 |
| 备选3 | `yunshancase.cc` | 不含年份，更通用 |

### 域名选择原则

- 优先短域名（<15字符）
- 包含「shanyue」或「2015」中至少一个，确保与游戏名的关联
- 避免中文拼音全拼过长（如 `shanyuewushengyunshanyizhong` 不可取）
- .cc 或 .ink 作为首选后缀（便宜且不常见，有辨识度）

## 品牌调性

### 关键词

| 是 | 不是 |
|----|------|
| 克制 | 煽情 |
| 沉静 | 沉闷 |
| 真实感 | 猎奇 |
| 档案感 | 恐怖感 |
| 社会关怀 | 说教 |
| 怀旧 | 复古卖弄 |

### 表达原则

1. **不消费苦难**：游戏涉及校园霸凌、制度性腐败、少年死亡等沉重题材，品牌表达要保持尊重和克制，不用耸动的措辞吸引眼球
2. **不剧透核心反转**：所有对外宣传材料只透露阶段一的信息（一个少年的死亡），不提及高考移民、陈默的真实身份、系统性腐败等后期内容
3. **档案感优先**：品牌视觉和文字都应该有「翻阅旧档案」的质感 -- 冷静、客观、留白，让玩家自己去感受和判断
4. **琥珀色的克制使用**：琥珀色代表「灯火/希望」，只在关键位置出现。如果到处都是琥珀色，就失去了「黑暗中的一点光」的意义
